| 項目 | 仮定 |
|---|---|

| 住居 | 70平米前後・最寄り駅徒歩10分以内・ペット可（柴犬2匹）・賃貸 |

### 1-2. 6軸スコアリング基準（各10点満点・重み付き100点）

| 軸 | 重み | 10点の条件 |
|---|---|---|
| ①コスト | 25% | 年350万円以下=10／400万=8／450万=6／500万=4／500万超=2（線形補間） |
| ②教育環境 | 20% | 施設数・学力・待機児童・費用支援を総合評価 |
| ③公園・ドッグラン | 15% | 大型公園の数と面積、ドッグラン充実度 |
| ④夏の涼しさ | 15% | 8月平均最高気温：≤26℃=10／≤28=8／≤30=6／≤32=4／32超=2 |
| ⑤生活利便性・治安 | 15% | スーパー・病院密度、犯罪率、生活インフラ |
| ⑥行政サービス | 10% | 移住支援金・医療費助成・保育料軽減 |

### 1-3. 年間コスト算出（v2修正版）

> v1で家賃+食費+光熱+交通+「その他150万」の合算ミスがあり、新候補9市すべてで年合計を再計算。

| 費目 | 算出基準 |
|---|---|
| 住居費 | 実勢賃貸相場の中央値×12ヶ月（駐車場込み） |
| 食費 | 4人世帯標準値×地域物価補正（東京比 -5〜-10%） |
| 光熱費 | 寒冷地36-40万（暖房費込み）／温暖地25-31万 |
| 交通費 | 車1台年55万円（保険・税・ガソリン・車検均等） |
| その他150万 | 教育20＋医療5＋ペット25＋雑費100 |


- **暖房費**は年12〜22万円（高断熱物件選定で抑制可能）。
- 中2体力で全国45-47位。冬季屋外活動の構造的制約。
- 移住支援金が令和7年度から減額（世帯100万→60万円・子1人加算10万円のみ）。

### 盛岡市
- **中3数学が岩手県43位（全国比-5pt）**。中学生のいる家庭は塾選びが重要。
- 賃料が八戸・帯広・富山よりやや高い。

### 富山市
- 8月平均最高31.4℃と暑い。湿度高くフェーン現象あり。
- 冬の積雪は新潟より多い年もある。
- 呉羽山断層帯（M7想定）の直下型地震リスク。

### 仙台市
- 地震リスク：東日本大震災の震源域。耐震・液状化対策物件選定必須。
- 仙台駅周辺（青葉区）は犯罪件数が多い。**泉区・太白区**を推奨。
- 移住支援金の申請には東京圏からの移住要件あり。

### 金沢市
- 通院助成が中3まで（入院は18歳）と医療費助成が他都市より弱い。
- 8月平均最高31.3℃。**2024年は史上初の月平均30℃超**。
- 能登半島地震2024年で震度5強の影響。冬の日照時間は東京の半分以下。

### 松本市・長野市
- 長野県は移住支援金が令和8年度以降「子加算30万円」に減額予定の自治体あり。早期申請推奨。
- 松本は特急あずさで新宿2時間40分、月数回出社が限度。

### 北九州市
- **8月平均最高32.5℃**と暑い（v2修正）。PM2.5・黄砂飛来あり。
- 公的無料ドッグラン専用施設は限定的。
- 治安は改善傾向だが小倉北区中心部は他区より発生率高め。**八幡西区・小倉南区**を推奨。
- 都内出社は実質的に四半期1回（年130万円）。

### 新潟市
- **夏の暑さは8月平均30.6℃**。フェーン現象で40℃台の可能性。
- 学力テストは全国平均以下（算数・数学）。
- 車社会であり、公共交通のみでの生活は困難。

### 軽井沢町
- **年間コスト差が最大**（新潟比+229万円・10年で+2,290万円）。
- **冬の小児科は平日のみ**。重篤時は佐久医療センター（車30分）が後方病院。
- 通年賃貸物件が市場に少なく、柴犬2匹可の物件確保が難しい。
- UIJターン支援金が対象外。

### 花小金井
- 年間猛暑日が2024年は35日。子どもの外遊び・犬の散歩が夏季に大きく制限される。
- 教員1人当たり17.10人と全国最多負担。
- ただし**学力テスト・難関大進学パイプは全国最強**で、教育投資を最大化したい家庭には最適。

## 7. 推奨ロードマップ（家族タイプ別・v2）

### 7-1. 「夏涼しさ＋コスト両立」を最優先
→ **帯広（74.0）→ 八戸（74.0）→ 函館（73.5）→ 札幌（72.0）**
- 帯広：14都市で最安・最涼。-20℃級と医療費中3までが弱点。
- 八戸：移住支援300万＋医療費18歳＋小学校学力上位。バランス。
- 函館：札幌より暖房費安・空港至近。**都内出社は四半期1回限度**。
- 札幌：政令市の利便性・中3で全国超の学テ。

### 7-2. 「リモート×月数回都内出社」を最優先
→ **仙台（70.5・年50万）→ 盛岡（71.5・年70万）→ 長野（65.0・年45万）**
- 仙台：政令市学力トップ・移住支援300万・新幹線1時間40分。
- 盛岡：新幹線2時間10分・全国学テ小6国語上位。中3数学43位の塾依存リスク。
- 長野：新幹線1時間20分で東京最速・ドッグラン少。

### 7-3. 「学力（特に上位層の進学）」を最優先
→ **花小金井（教育統合75）→ 仙台市・札幌市（67）→ 金沢市（教育9点）**
- 花小金井：難関大進学パイプは圧倒的（東大1081・東京都全体4教科+2.2〜+4.6pt）。
- 仙台市：政令市単独学テで中3数学+3pt・国語+3pt。**仙台二高が東北大78名**。
- 金沢市：石川県は全国学テ4教科全国超。**金沢泉丘高校が東大23・京大34**。

### 7-4. 「犬と自然の豊かさ」を最優先
→ **松本（68.0）→ 帯広（74.0）→ 軽井沢（56.5）**
- 松本：信州スカイパーク無料ドッグラン・アルプス公園71ha。
- 帯広：十勝エコロジーパーク・牧場文化。
- 軽井沢：自然と特色教育の頂点だが年690万円〜の覚悟が必要。

### 7-5. 「行政サービス手厚さ」を最優先
→ **仙台（70.5）／新潟（62.5）／北九州（64.5）**
- 仙台・新潟：移住支援金最大300万円。
- 北九州：移住支援250万＋待機児童12年連続ゼロ＋医療費18歳所得制限なし。

## 8. 次のアクション（移住検討のステップ）

### Step 1：絞り込み（今すぐ）
- 「夏涼しさ／コスト／教育／都内出社の現実性」で家族の優先順位3つを確定
- 上位3-5候補を絞る

### Step 2：体験移住（2〜3ヶ月以内）

| 都市 | 体験移住制度 |
|---|---|
| 帯広 | 十勝移住相談センター／お試し住宅（家具付き） |
| 八戸 | 八戸市移住相談窓口／お試し住宅あり |
| 函館 | 函館市移住相談窓口／お試し体験住宅 |
| 札幌 | さっぽろ移住ポータルで相談可 |
| 盛岡 | 盛岡市UIJターン相談窓口 |
| 富山 | 富山市移住相談窓口／お試し短期住居 |
| 仙台 | 問い合わせ窓口あり |
| 金沢 | 金沢市移住相談デスク |
| 松本 | 松本市移住・定住推進課 |
| 長野 | 長野市Iターン相談窓口 |
| 北九州 | 北九州市住みよかネット移住相談 |
| 新潟 | 体験居住支援：1週間以上で世帯5万円 |
| 軽井沢 | 「おためし移住」（要問い合わせ） |

### Step 3：ハザード／医療／学校確認
- 各市のハザードマップで物件候補地の津波・水害・液状化リスク確認
- 夜間小児科・三次救急までの実所要時間（カーナビで実測）
- 候補の中学校学区の進学実績（市町村教委PDFで公表あり）

### Step 4：賃貸内見＋柴犬2匹可物件確保
- 柴犬2匹可は「複数頭可」「大型犬可」でフィルタリング（敷金1-2か月追加が一般的）
- 軽井沢：地元不動産（軽井沢商事等）への直接問い合わせ必須

### Step 5：移住支援金の申請（転入後1年以内）
- 申請には東京圏からの移住要件（直前1年以上の居住・就業確認）
- 子2人帯同で最大300万円。申請手続きを転入前に確認
