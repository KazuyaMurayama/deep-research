## 第3章 Wemofインジケーターの実態分析

### 3.1 WEMOF 概要

**WEMOF = Wakkyai's Entry Model Of FX**の頭字語。わっきゃい氏が自身の逆張りエントリー判断を半自動化するために開発したカスタムインジケーター（TradingView対応）。3つの古典的テクニカル指標の組み合わせで構成され、固有の独自数式は確認されていない。

| 指標名 | 役割 | Wemofでの使い方 |
|--------|------|----------------|
| ボリンジャーバンド ±3σ（5分足ドル円） | 統計的な価格異常値検出 | ±3σタッチを過剰反応の合図として逆張りエントリーシグナルに使用 |
| %B（パーセントB） | バンド内価格相対位置を0〜1で数値化 | ±3σタッチをノイズと区別する価格位置フィルター |
| ピボットポイント | 前日高値・安値・終値から算出するサポレジ | ±3σ反発とピボット水準の重複を補助確認するフィルター |

### 3.2 エントリー条件と見送りフィルター

| 項目 | 条件 |
|------|------|
| 主な通貨ペア | USD/JPY（ドル円）、XAU/USD（金） |
| 時間軸 | 主に5分足 |
| 1日の利益目標 | 10pips |
| 利確幅 | 2〜3pips |
| 損切幅 | 50〜100pips（逆張りのため広めに設定） |

**見送りフィルター：** 経済指標発表前後 / 直近高値・安値ブレイク中 / ダブルトップ等パターン完成時 / キリ番での反応局面

### 3.3 リスクリワード比の構造的問題

```
利確幅 : 損切幅 = 2〜3pips : 50〜100pips → RR ≒ 1:16.7 〜 1:50
```

| 利確 / 損切 | RR | 損益分岐勝率 |
|------------|-----|-------------|
| 3pips / 50pips | 1:16.7 | **約94.4%** |
| 3pips / 100pips | 1:33.3 | **約97.1%** |
| 2pips / 50pips | 1:25 | **約96.2%** |
| 2pips / 100pips | 1:50 | **約98.0%** |

**1回の負けで30〜50回分の勝ちを失う構造**。ナシーム・ニコラス・タレブ（Nassim Nicholas Taleb）が指摘する「ファットテール戦略の典型」であり、平時には連勝が積み上がる代わりにテールイベントで一気に元本毀損する。

#### 損益期待値（EV）シミュレーション（利確3pips／損切50pips 基本ケース）

| 勝率 | 期待値（pips/trade） | 100トレード後の理論損益 |
|------|---------------------|-----------------------|
| 93% | 3×0.93 - 50×0.07 = **-0.71pips** | -71pips（損失） |
| 95% | 3×0.95 - 50×0.05 = **+0.35pips** | +35pips |
| 97% | 3×0.97 - 50×0.03 = **+1.41pips** | +141pips |
| 99% | 3×0.99 - 50×0.01 = **+2.47pips** | +247pips |

**勝率93%を下回ると即座にマイナス期待値**となる。本人主張の「30連勝当たり前」は理論上勝率96.7%以上に相当するが、プロでも維持困難な水準であり長期再現は極めて難しい。
