# GMOコイン 入金合計額の確認方法ガイド
### 〜 2020年から投資してきた分を全部チェックしよう 〜

**作成日：** 2026年5月1日  
**対象：** GMOコインで2020年8月〜2026年にかけて投資していた方  
**難易度：** ★☆☆☆☆（高校生でもできる！）

---

## はじめに：なぜ「入金額」を確認するの？

2020年から積み上げてきたビットコインなどを **2026年に現金化（売却）** した場合、
税金の計算が必要になります。

そのとき一番大事なのが

> **「いくら入金して、いくら引き出したか（利益はいくらか）」**

の把握です。入金履歴をまとめておくと、確定申告のときにとても助かります。

---

## 全体の流れ（3ステップ）

```
STEP 1：GMOコインにログイン
    ↓
STEP 2：入金履歴ページを開く
    ↓
STEP 3：CSV（エクセルファイル）をダウンロードして合計を出す
```

---

## STEP 1：GMOコインにログイン

### パソコン（おすすめ）の場合

1. ブラウザで **GMOコイン公式サイト** を開く
2. 右上の「**ログイン**」ボタンをクリック
3. 登録したメールアドレスとパスワードを入力
4. 二段階認証（SMS or 認証アプリ）でコードを入力

> **ポイント：** スマホアプリより **パソコンのブラウザ版** のほうが、
> 履歴のダウンロードがしやすくておすすめです。

---

## STEP 2：入金履歴ページを開く

ログイン後、画面左側（またはメニュー）から操作します。

### ＜日本円の入金履歴を確認する場合＞

```
メニュー ▶ 「入出金」または「資産」 ▶ 「入出金履歴」
```

| 操作 | 内容 |
|------|------|
| ① 上部メニューの「資産管理」をクリック | |
| ② 左サイドバーの「**入出金履歴**」を選択 | |
| ③ 「**日本円**」タブを選択 | 仮想通貨ではなく「円」のタブ |
| ④ 期間を「**2020年8月〜2026年**」に設定 | 開始日：2020/08/01、終了日：今日 |
| ⑤ 「検索」ボタンをクリック | |

> **注意：** 1回の検索で出せる期間に上限（例：1年分ずつ）がある場合があります。
> その場合は **1年ごとに分けて検索** してください（後で合計します）。

---

## STEP 3：CSVをダウンロードして合計を出す

### CSVって何？

**CSV（シーエスブイ）** ＝ エクセルで開けるデータファイルのことです。
数字がきれいに並んでいるので、合計が簡単に計算できます。

### ダウンロード手順

```
履歴一覧の上 or 下にある「CSVダウンロード」ボタンをクリック
→ ファイルが自動でパソコンに保存される
```

### エクセル（またはGoogleスプレッドシート）で合計する

1. ダウンロードしたCSVファイルを開く
2. 「入金額」や「金額」の列を探す
3. その列の一番下の空白セルに以下を入力

```
=SUM(C2:C1000)   ← C列が金額列の場合
```

4. Enterを押すと **合計入金額** が表示される！

> **複数年分ある場合：** 年ごとのCSVをすべて開いて、
> それぞれの合計を足し算すればOKです。

## スマホアプリで確認する場合

パソコンが使えないときはアプリでも確認できます。ただし **CSVダウンロードはできない** ことが多いので、目安の確認用に使いましょう。

```
アプリ起動 ▶ 下メニュー「資産」 ▶ 「入出金履歴」 ▶ 「円」を選択
```

## よくあるトラブルQ&A

### Q. 履歴が全部出てこない
**A.** 期間指定の「開始日」が間違っている可能性があります。
2020年8月1日（2020/08/01）になっているか確認してください。

---

### Q. 入金と出金が混ざっている
**A.** 「種別」や「区分」の列を見てください。
「**入金**」と書いてあるものだけを合計すればOKです。
エクセルの「フィルター機能」を使うと便利です。

---

### Q. 仮想通貨（ビットコインなど）の入金も出てくる
**A.** それは別物です。
今回チェックしたいのは **「日本円」** の入金だけです。
「仮想通貨」タブではなく **「日本円」タブ** を選んでいるか確認しましょう。

---

### Q. CSVダウンロードボタンが見当たらない
**A.** ブラウザを **Google Chrome** に変えてみてください。
また、ページを一番下までスクロールすると出てくることもあります。

## 税金のこと：ちょっとだけ知っておこう

2020年から2026年にかけてビットコインなどで利益が出た場合、**確定申告が必要** になる可能性があります。

| 項目 | 内容 |
|------|------|
| 利益の種類 | 雑所得（ざっしょとく） |
| 計算式 | **売却額 − 取得費 − 手数料 = 利益** |
| 申告が必要な目安 | 年間利益が **20万円超** の場合（会社員の場合） |
| 申告時期 | 翌年2月16日〜3月15日 |

> **入金額（＝投資元本）** を把握しておくと、税理士さんや確定申告書類の作成に役立ちます。
