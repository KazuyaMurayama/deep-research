# GMOコイン 入金合計額の確認方法ガイド
### 〜 2020年から投資してきた分を全部チェックしよう 〜

**作成日：** 2026年5月1日  
**対象：** GMOコインで2020年8月〜2026年にかけて投資していた方  
**難易度：** ★☆☆☆☆（高校生でもできる！）

---

## はじめに：なぜ「入金額」を確認するの？

2020年から積み上げてきたビットコインなどを **2026年に現金化（売却）** した場合、
税金の計算が必要になります。

そのとき一番大事なのが

> **「いくら入金して、いくら引き出したか（利益はいくらか）」**

の把握です。入金履歴をまとめておくと、確定申告のときにとても助かります。

---

## 方法は2つ。まずどちらか選ぼう

| 方法 | 特徴 | おすすめ度 |
|------|------|-----------|
| **方法A：年間取引報告書を使う** | 1年分がPDF1枚にまとまっている。一番かんたん | ★★★ おすすめ |
| **方法B：入出金履歴から直接確認する** | 日付・金額を自分でExcelに集める | ★★ 自分でまとめたい人向け |

> **迷ったら方法Aから試してください。** 2020年〜2025年分の報告書を揃えるだけで入金額がすぐわかります。

---

## 方法A：年間取引報告書を使う（一番かんたん）

### 年間取引報告書って何？

GMOコインが毎年1月中旬に自動で作ってくれる **PDF書類** です。
1年分の入出金・取引が1枚にまとまっていて、確定申告にもそのまま使えます。

### 手順（パソコン推奨）

**① GMOコインにログイン**

1. ブラウザで `https://coin.z.com/jp/` を開く
2. 右上「ログイン」をクリック
3. メールアドレス・パスワードを入力
4. 二段階認証コードを入力

---

**② 「帳票」ページを開く**

ログイン後、**左側のメニュー**から操作します。

```
左メニュー「明細」の横にある「＋」をクリック
     ↓
「帳票」をクリック
```

---

**③ 年間取引報告書をダウンロードする**

1. 期間の「受信日」を確認したい年に設定（例：2020年1月〜12月）
2. 一覧に「**年間取引報告書**」が表示される
3. クリックして **PDF をダウンロード**

> **2020年〜2025年分まで、1年ずつ繰り返します**（全部で最大6回）

---

**④ 入金合計を確認する**

ダウンロードしたPDFを開くと、**「日本円入金額」の合計** が記載されています。

年ごとの金額を以下の表に書き留めておきましょう：

| 年 | 入金合計額（円） | メモ |
|----|----------------|------|
| 2020年（8月〜12月） | ¥　　　　　　 | |
| 2021年 | ¥　　　　　　 | |
| 2022年 | ¥　　　　　　 | |
| 2023年 | ¥　　　　　　 | |
| 2024年 | ¥　　　　　　 | |
| 2025年 | ¥　　　　　　 | |
| 2026年（1月〜現在） | ← 方法Bで確認 | まだ報告書が出ていない |
| **合計** | **¥　　　　　　** | |

> **⚠️ 2026年分はまだ報告書が出ていません（翌年1月以降）。**  
> 2026年分だけは後述の「方法B」で確認してください。

---

## 方法B：入出金履歴から直接確認する

2026年分の確認や、細かい入金日・金額を自分でチェックしたい場合に使います。

### 手順（パソコン推奨）

**① GMOコインにログイン**（方法Aと同じ）

---

**② 「日本円の入出金履歴」ページを開く**

```
左メニュー「入出金」をクリック
     ↓
「日本円」をクリック
     ↓
「取引履歴」をクリック
```

---

**③ 期間と区分を設定して検索する**

| 設定項目 | 入力内容 |
|----------|---------|
| 期間（開始） | 2026/01/01 |
| 期間（終了） | 今日の日付 |
| 入出金区分 | 「入金」のみ選択（または「すべて」） |

「**検索**」ボタンをクリック → 一覧が表示される

---

**④ CSVをダウンロードして合計する（推奨）**

画面に表示された履歴は目視でも確認できますが、**CSVダウンロード**を使うと合計が楽です。

CSVダウンロードの手順：

```
左メニュー「明細」の横にある「＋」をクリック
     ↓
「CSV」をクリック
     ↓
確認したい「年度」の「CSVを作成する」をクリック
     ↓
確認ポップアップで「実行」をクリック
     ↓
（しばらく待つ）ページを更新する
     ↓
「CSVダウンロード」ボタンをクリック
     ↓
ZIPファイルが保存される → 解凍する
```

> **⚠️ ファイルはZIP形式で届きます。**  
> Windowsなら右クリック→「すべて展開」で解凍できます。

---

**⑤ エクセルで合計を出す**

1. 解凍したCSVファイルを **Excel または Googleスプレッドシート** で開く
2. 「入金額」や「金額」の列を探す
3. 入金だけを合計する

```excel
=SUMIF(B列, "入金", C列)   ← B列が「区分」、C列が「金額」の場合
```

または「フィルター」機能で「入金」のみに絞って

```excel
=SUM(C2:C1000)
```

## スマホアプリで確認する場合

> **注意：** アプリでもCSVダウンロードは可能ですが、操作がパソコンより少し複雑です。  
> 入金額の確認だけなら、 **年間取引報告書（方法A）をパソコンで確認する** のが最も楽です。

スマホでの操作（確認用）：

```
アプリ起動
     ↓
画面左上の「三本線（ハンバーガーメニュー）」をタップ
     ↓
「入出金」→「日本円」をタップ
     ↓
「取引履歴」をタップ
     ↓
期間・入出金区分を設定して「検索」
```

CSVダウンロード（スマホ）：

```
三本線メニュー → 「明細」 → 「CSV」
→「CSVを作成」→「実行」→「CSVダウンロード」
```

## よくあるトラブルQ&A

### Q. 年間取引報告書が見当たらない
**A.** 報告書は **毎年1月中旬以降** に届きます。2020〜2025年分はすでに届いているはずです。  
「帳票」一覧の期間設定を広げて（受信日を「年間」に）再検索してみてください。

---

### Q. CSVを開いたら文字化けした
**A.** GMOコインのCSVはExcelで直接開くと文字化けすることがあります。

対処法：
1. Excelの「データ」タブ → 「テキストファイル」からインポート
2. 文字コードを「**UTF-8**」に設定してインポート

または、ファイルを **メモ帳にドラッグ＆ドロップ** して確認できます。

---

### Q. 2020年より前の履歴は確認できる？
**A.** 2020年開始なので問題ありません。ただし **2017年以前の取引はCSVダウンロード不可** のため、
もし2017年以前にも取引があった場合は「取引残高報告書」を別途取得する必要があります。

---

### Q. 入出金が混ざって「入金だけ」の合計がわからない
**A.** ExcelやGoogleスプレッドシートの **フィルター機能** を使います。

1. 列の見出し行をクリック → 「データ」→「フィルター」
2. 「区分」列の▼をクリック → 「入金」だけにチェック
3. 表示されている金額をSUMで合計

---

### Q. 仮想通貨での入金（ビットコインを他の取引所から送付）も含める？
**A.** 「日本円でいくら入金したか」を確認する場合は **日本円入金のみ** でOKです。  
仮想通貨の入金（送付）は別の帳票（「入出金」→「暗号資産」）に記録されています。

## 税金のこと：ちょっとだけ知っておこう

2026年にビットコインなどを現金化した場合、**確定申告が必要** になる可能性があります。

| 項目 | 内容 |
|------|------|
| 利益の種類 | 雑所得（ざっしょとく） |
| 計算式 | **売却額 − 取得費 − 手数料 = 利益** |
| 申告が必要な目安 | 年間利益が **20万円超** の場合（会社員の場合） |
| 申告時期 | 翌年2月16日〜3月15日 |
| GMOコインの提供ツール | **年間取引報告書**（明細→帳票）、損益計算ツール（明細→精算表）|

> **GMOコインには「損益計算ツール」もあります。**  
> 「明細」→「精算表」から確認でき、移動平均法・総平均法どちらでも計算できます。  
> これを使えば「利益がいくらか」もかんたんに出せます。

## まとめ：やることリスト

**方法A（年間取引報告書）で入金合計を確認する場合：**

- [ ] パソコンでGMOコインにログイン
- [ ] 左メニュー「明細」→「＋」→「帳票」を開く
- [ ] 2020年〜2025年の「年間取引報告書」を1年ずつダウンロード（PDF）
- [ ] 各PDFの「日本円入金額」を上の表に記録
- [ ] 2026年分は「方法B（入出金履歴）」で補完
- [ ] 全年分を合計する

**方法B（CSVダウンロード）で細かく確認する場合：**

- [ ] 左メニュー「明細」→「＋」→「CSV」を開く
- [ ] 確認したい年の「CSVを作成する」→「実行」→「CSVダウンロード」
- [ ] ZIPファイルを解凍してExcelで開く（文字化けはUTF-8設定で解決）
- [ ] フィルターで「入金」のみに絞ってSUM関数で合計
