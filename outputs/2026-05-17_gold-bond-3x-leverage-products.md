最終更新日: 2026-05-17

## 1. エグゼクティブサマリー

**要点：**
- ゴールド・ボンドへのレバレッジ3倍以上の手段は、NASDAQ-100系と比べて選択肢が**大きく限定される**
- ゴールド系で純粋に3倍以上のレバレッジをかけられる日本居住者向け商品は「SBI証券 金CFD（最大20倍）」「TOCOM金先物（限日取引）」に限定される。欧州上場の3GOLは日本から直接購入不可
- ボンド系の3倍商品で国内取引可能なのは「TMF」「TMV」（米国ETF）のみ。SBI証券の自社CFDには米国債系がない
- **くりっく株365 金は2024年12月に廃止済みのため利用不可**
- SOFR現在値：**3.56%**（2026年5月14日）。レバレッジETFの実質コストに直接影響する

### 推奨TOP3（ゴールド用）

| 順位 | 商品 | 実質レバレッジ | 最大の強み |
|------|------|--------------|-----------|
| 1位 | SBI証券 金CFD | 最大20倍（調整可能） | T+0即時決済、ボラドラッグなし、レバレッジ自在設定。コスト非透明が課題 |
| 2位 | TOCOM 金先物（限日取引） | 理論上の高倍率可能 | 取引所取引で透明性高、ロールコスト年率1〜3%程度、T+0現金決済 |
| 3位 | 1540 純金上場信託×信用買い | 約2倍（最大） | ボラドラッグなし、信用金利年率2.80%と低コスト。2倍止まりが難点 |

> ⚠️ **WisdomTree 3GOL（3倍）**は欧州取引所（LSE）上場のため、日本から直接購入不可。SBI証券でも取扱なし。UGL/JNUGは最大2倍であり3倍未達。

### 推奨TOP3（ボンド用）

| 順位 | 商品 | 実質レバレッジ | 最大の強み |
|------|------|--------------|-----------|
| 1位 | TMF（Direxion 20+ Year Treasury Bull 3X） | 3倍固定 | SBI証券で購入可、特定口座対応、株式損益通算可 |
| 2位 | TMV（Direxion 20+ Year Treasury Bear 3X） | 3倍固定（逆張り） | 金利上昇局面の逆ヘッジ手段。SBI証券で購入可 |
| 3位 | IG証券 米国30年国債先物CFD | 最大50倍 | 高倍率・T+0即時。ただしSBI証券では非取扱。コスト要確認 |

> ⚠️ **SBI証券の自社CFDには米国債系がない**（株価指数のみ）。債券3倍以上のレバレッジはTMF/TMVか他社CFDを利用する必要がある。

**3行まとめ：**
ゴールド3倍以上の手段として現実的なのは「SBI証券 金CFD（最大20倍）」と「TOCOM金先物」のみであり、欧州の3GOLは日本非対応。ボンド系は「TMF/TMV」（米国ETF・SBI証券取扱）が事実上唯一の選択肢だが、SOFR×2=7.12%相当のスワップコスト（推計）が重くのしかかる。いずれも国内NASDAQ系と異なり商品の絶対数が少なく、コスト・税制・受渡日数を精査したうえで商品を選択することが不可欠である。

## 2. 前提・用語定義

**要点：**
- ゴールド特有の金リース金利・XAUの概念を把握する
- ボラティリティドラッグは日次リバランス型ETFのみに発生する
- T+n受渡の違いが短期取引のコストに直接影響する

### 2-1. 基準指標・資産定義

| 用語 | 説明 |
|------|------|
| XAU/USD | ゴールドの国際価格（トロイオンス建て米ドル）。金CFD・UGL等が追う基準指数 |
| SOFR | 米国翌日物資金調達金利。2026年5月14日現在 **3.56%**。CFD・レバレッジETFの内部コストに直結 |
| 金リース金利（Gold Lease Rate / GOFO） | 金の貸借コスト。LBMA（ロンドン貴金属市場協会）は2015年に公式公表を廃止。現在は0.1〜0.5%程度と推計される |
| TLT | iShares 20+ Year Treasury Bond ETF（米国長期国債ETF）。TMF/TMVの参照原指数の一般的ベンチマーク |
| ボラティリティドラッグ（ボラドラッグ） | 日次リバランス型レバレッジETFに固有の構造的損失。D ≈ 0.5 × L × (L-1) × σ² で推計。CFD・先物には発生しない |
| T+n | 約定から受渡（決済完了）までの営業日数。T+0＝即日、T+2＝約定翌々営業日 |

### 2-2. くりっく株365 金の廃止

- 東京金融取引所（TFX）提供の「くりっく株365」金銘柄は、**2024年12月19日（取引最終日）・12月23日（最終決済）でサービス終了**
- 2025年以降の新規建ては不可。本レポートでは「廃止済み商品」として記録のみ行い、現行選択肢には含めない

### 2-3. 共通前提条件

本レポートのコスト試算は以下の条件を統一して使用する。

- 元本：100万円（参考ベース）
- 保有期間：1年（365日）
- 対象資産年間騰落：0%（横ばい。コスト比較に集中するため）
- SOFR：3.56%（2026年5月14日）
- 金の年率ボラティリティ（σ）：16.2%（WGC/BullionVault参考値）
- 米国長期国債（TLT）の年率ボラティリティ（σ）：12.5%（過去30年平均）
- 為替：1ドル＝参考値。為替リスクは別途考慮が必要

## 4. ゴールド系：各商品の詳細解説

### 4-1. SBI証券 金CFD（最大20倍レバレッジ）

**概要**

SBI証券が2025年8月25日より提供開始した店頭金CFD。XAU/USDをベースとする差金決済商品。

| 項目 | 内容 |
|------|------|
| 提供開始 | 2025年8月25日 |
| 最大レバレッジ | 20倍（証拠金率5%） |
| スプレッド | 非開示・変動制 |
| オーバーナイト金利 | 非公開（「カバー取引コスト反映」のみ） |
| 受渡日数 | T+0（差金決済即時） |
| ボラドラッグ | **なし**（日次リバランス型ではないため） |
| 税制 | 先物取引に係る雑所得等（申告分離20.315%） |
| 損益通算 | 国内CFD間のみ可。株式損益とは通算不可 |

**オーバーナイト金利の推計コスト構造**

SBI証券は具体的な計算式を非公開としているが、業界一般の金CFDは以下の構造を持つ。

```
オーバーナイト金利（買方負担） ≈ SOFR + 金リース金利 + 業者マージン
                            ≈ 3.56% + 0.3% [推計] + 0.5〜1.0% [推計]
                            ≈ 4.4〜5.0%/年 [推計]
```

- SOFR：3.56%（2026年5月14日）
- 金リース金利：LBMA廃止後は市場推計値0.1〜0.5%を使用（中央値0.3%で計算）[推計]
- 業者マージン：各社によって異なる。0.5〜1.0%程度と推計 [推計]

> ⚠️ **コスト非透明性が最大の課題**。SBI証券は具体的な計算式・レートを公表していないため、上記はあくまで業界標準に基づく推計値である。実際の保有コストは取引後の口座明細で確認するしかない。

**活用シナリオ**

- 短期（数日〜数週間）の金価格方向性取引に向く
- 3倍以上（例：20倍設定）のレバレッジは、1%の逆行で証拠金の20%を失うため、ストップロスの設定が必須
- T+0決済のため緊急時の即時手仕舞いが可能

### 4-2. UGL（ProShares Ultra Gold・2倍）

**概要**

ProShares社が提供する金2倍レバレッジETF。先物（約86%）とスワップ契約（Citi/UBS/GS等）で運用。

| 項目 | 内容 |
|------|------|
| ティッカー | UGL（NYSEArca上場） |
| レバレッジ | 2倍（3倍未達） |
| 経費率 | 0.95%/年 |
| スワップ埋込コスト | SOFR×(2-1) = 3.56%×1 = 3.56%/年 [推計] |
| 実質年率コスト | 0.95% + 3.56% = **4.51%/年** [推計] |
| ボラドラッグ | D = 0.5×2×1×0.162² ≈ **2.62%/年** （σ=16.2%で試算） |
| SBI証券取扱 | ○（米国ETF） |
| NISA対応 | **不可**（レバレッジETFは新NISA対象外） |
| 受渡日数 | T+2（日本での受渡。米国現地はT+1化済み） |
| 税制 | 株式等（申告分離20.315%） |
| 損益通算 | **可**（株式損益と通算可能） |

**スワップ埋込コストの根拠**

レバレッジETFは原資産の(L-1)倍相当のスワップ契約を保有する。この契約に付随するファイナンスコストは概ねSOFRに連動する。

```
スワップ埋込コスト ≈ SOFR × (レバレッジ - 1)
                  ≈ 3.56% × (2 - 1) = 3.56%/年 [推計]
```

> ⚠️ **3倍未達（2倍止まり）**。純粋な金3倍レバレッジを求める場合、UGLのみでは不足。信用取引で追加レバレッジをかけることも理論上可能だが、信用金利コストが追加発生する。

### 4-3. JNUG（Direxion Daily Junior Gold Miners Bull 2x ETF）

**概要**

Direxion社が提供する「ジュニア金鉱会社株指数（MVIS Global Junior Gold Miners Index）」の2倍連動ETF。

| 項目 | 内容 |
|------|------|
| ティッカー | JNUG（NYSEArca上場） |
| レバレッジ | 2倍（3倍未達） |
| 経費率 | 1.03%/年（実質上限0.95%、2027年9月まで） |
| スワップ埋込コスト | SOFR×1 = 3.56%/年 [推計] |
| 実質年率コスト | 1.03% + 3.56% = **4.59%/年** [推計] |
| SBI証券取扱 | ○（米国ETF） |
| NISA対応 | **不可** |
| 受渡日数 | T+2 |

**⚠️ 最重要注意点：金現物とは別物**

JNUGは金地金（現物金）の価格に連動するものでは**ない**。

- 連動対象：金鉱会社（特に時価総額の小さい「ジュニア」鉱山会社）の株式指数
- ジュニア金鉱株は金価格に正の相関を持つが、株式固有のリスク（会社倒産・経営失敗・地政学リスク等）を追加で負う
- ボラティリティは金現物より大幅に高く、金価格が上昇しても鉱山会社の業績悪化で下落することがある
- 「金に投資したい」という目的では**JNUG は不適切**

### 4-4. 1540 純金上場信託×信用買い

**概要**

三菱UFJアセットマネジメント（旧MUFG）が提供する純金ETF。信用取引を利用することで約2倍のレバレッジが可能。

| 項目 | 内容 |
|------|------|
| ティッカー | 1540（東証上場） |
| ETF自体のレバレッジ | ×1（現物連動） |
| 信用買いでの実質レバレッジ | 最大約2倍（SBI証券・制度信用） |
| 信用買い金利 | **年率2.80%**（SBI証券・制度信用） |
| 信託報酬 | 0.44%（税込） |
| 実質年率コスト（信用買い時） | 2.80% + 0.44% = **3.24%/年** [推計] |
| ボラドラッグ | **なし**（現物連動ETFのため） |
| SBI証券取扱 | ○（制度信用・一般信用対応） |
| NISA対応 | 現物購入は○（新NISA対象）。信用取引はNISA非対応 |
| 受渡日数 | T+2 |
| 税制 | 株式等（申告分離20.315%） |
| 損益通算 | 可（株式損益と通算可能） |

**活用シナリオ・限界**

- コスト最安クラス（信用金利2.80%は金CFDのオーバーナイト金利推計4.4〜5%より低い）
- ボラドラッグがなく中長期保有向き
- 最大レバレッジが約2倍止まりのため、3倍以上を求める場合は不適切

### 4-5. TOCOM 金先物（限日取引）

**概要**

大阪取引所（OSE）が運営するTOCOM（東京商品取引所）の金先物。SBI証券では「金限日取引（現金決済）」を取り扱う。

| 項目 | 内容 |
|------|------|
| 取引所 | 大阪取引所（TOCOM） |
| 商品規格 | 標準：1枚＝1kg、ミニ：1枚＝100g |
| SBI証券取扱 | ○（金限日取引・現金決済） |
| 決済方式 | 現金決済（限日取引は毎日ロール） |
| ロールコスト | **年率1〜3%程度** （コンタンゴ・金利環境依存） [推計] |
| 税制 | 先物取引に係る雑所得等（申告分離20.315%） |
| 損益通算 | 国内先物・CFD間で可。株式との通算不可 |
| ボラドラッグ | **なし**（先物取引のため） |
| 受渡日数 | T+0（現金決済即時） |

**限日取引の仕組み**

通常の先物と異なり、限日取引は毎日自動的にロール（翌限月への乗り換え）が発生する。これにより長期保有が可能だが、コンタンゴ（期先高）環境下ではロールコストが毎日発生する。

> ⚠️ 証拠金要件・追証（マージンコール）リスクは取引所先物と同様。高レバレッジ設定時の急落では証拠金が不足して強制ロスカットとなる可能性がある。

### 4-6. くりっく株365 金（廃止済み）

**概要（記録のみ）**

東京金融取引所（TFX）が提供していた取引所取引の金CFD商品。

- 取引最終日：**2024年12月19日**
- 最終決済日：**2024年12月23日**
- **2025年以降は新規建て不可。現在は利用できない**

かつては透明性が高い取引所取引の金レバレッジ商品として注目されていたが、廃止により現在選択肢には存在しない。

## 5. ボンド系：各商品の詳細解説

### 5-1. TMF（Direxion Daily 20+ Year Treasury Bull 3X）

**概要**

ICE U.S. Treasury 20+ Year Bond Indexの日次3倍連動ETF。長期米国国債の3倍ブル商品として個人投資家に広く利用されている。

| 項目 | 内容 |
|------|------|
| ティッカー | TMF（NYSE上場） |
| レバレッジ | 3倍（ブル） |
| 経費率 | 0.95%/年（上限キャップ2026年9月まで） |
| スワップ埋込コスト | SOFR×(3-1) = 3.56%×2 = **7.12%/年** [推計] |
| 実質年率コスト | 0.95% + 7.12% = **8.07%/年** [推計] |
| ボラドラッグ | D = 0.5×3×2×0.125² = **4.69%/年** （TLT σ=12.5%で試算） |
| SBI証券取扱 | ○（NYSE上場・外国株口座・特定口座対応） |
| NISA対応 | **不可**（レバレッジETFは新NISA対象外） |
| 受渡日数 | T+2（日本での受渡。米国現地はT+1） |
| 税制 | 株式等（申告分離20.315%） |
| 損益通算 | **可**（株式損益と通算可能） |

**スワップ埋込コストの重要な開示**

TMFのプロスペクタスには「スワップファイナンシングコストは経費率キャップの算定から除外される」と明記されている。つまり、表面上の経費率0.95%にはスワップコストが含まれていない。

```
スワップ埋込コスト ≈ SOFR × (レバレッジ - 1)
                  ≈ 3.56% × (3 - 1) = 7.12%/年 [推計]

実質コスト合計   = 経費率 + スワップ埋込
                 = 0.95% + 7.12% = 8.07%/年 [推計]
```

**金利リスクの二重化に注意**

TMFは長期米国国債に3倍レバレッジをかけるため：

1. **金利上昇リスク**：長期金利が上昇すると国債価格は下落→TMFは3倍の下落
2. **スワップコスト上昇リスク**：金利上昇（SOFR上昇）→スワップコストが増加

すなわち、金利上昇局面ではTMFはパフォーマンスの悪化と保有コストの増大が同時に発生する二重苦に直面する。

## 11. ボラティリティドラッグ解説

**ボラティリティドラッグ（ボラドラッグ）** は、日次リバランス型レバレッジETFに固有の構造的な損失で、複利効果の逆作用として発生する。

### 11-1. 計算式

```
D ≈ 0.5 × L × (L-1) × σ²

D = ボラティリティドラッグ（年率）
L = レバレッジ倍率
σ = 原資産の年率ボラティリティ（小数表示）
```

### 11-2. 各商品のボラドラッグ試算

| 商品名 | レバレッジ（L） | 年率ボラ（σ） | ボラドラッグ（D） | 備考 |
|--------|-------------|------------|----------------|------|
| SBI金CFD（任意倍率） | - | - | **0%** | CFDは日次リバランス型ではない |
| TOCOM金先物 | - | - | **0%** | 先物は日次リバランス型ではない |
| UGL（金2倍ETF） | 2 | 16.2% | 0.5×2×1×0.162² ≈ **2.62%/年** | 金σ=16.2%で試算 |
| JNUG（金鉱株2倍ETF） | 2 | > 16.2% | 2.62%以上（金より高ボラ） [推計] | 金鉱株は金より高ボラ |
| 1540×信用買い | - | - | **0%** | 現物ETFのため |
| WisdomTree 3GOL（3倍） | 3 | 16.2% | 0.5×3×2×0.162² ≈ **7.87%/年** [推計] | 日本購入不可だが参考試算 |
| TMF（米国債3倍） | 3 | 12.5% | 0.5×3×2×0.125² = **4.69%/年** | TLT σ=12.5%で試算 |
| TMV（米国債3倍ベア） | 3 | 12.5% | 同上 ≈ **4.69%/年** | |

### 11-3. ボラドラッグの直感的理解

```
【例：金2倍ETF（σ=16.2%）で1年間保有・横ばい（0%）のケース】

Day1: 金+10% → ETF+20%（100万→120万）
Day2: 金-10% → ETF-20%（120万→96万）

ベンチマーク（金）：100×1.10×0.90 = 99万円（-1%）
ETF実績：96万円（-4%）
差分：-3%（ボラドラッグ＝-3%程度）
```

- 金価格が**横ばい（往復ゼロ）**でもETFは損失が発生する
- ボラドラッグは市場の上下動が激しいほど大きくなる
- **長期保有ほどボラドラッグは累積する**（複利で悪化）
- CFD・先物にはこの問題が発生しない（日次リバランスをしないため）

### 11-4. 金利上昇局面でのボラドラッグ増大

2022年のFRB利上げ局面では、TLT（長期国債）のボラティリティが過去30年平均12.5%から推定15〜18%まで上昇した。

```
TMFボラドラッグ（σ=18%時）= 0.5×3×2×0.18² = 9.72%/年 [推計]

通常時（σ=12.5%）の 9.72% ÷ 4.69% ≈ 2.1倍のドラッグ増大
```

金利上昇局面でTMFを保有すると：（1）価格下落 + （2）スワップコスト増大 + （3）ボラドラッグ増大、の三重苦になる。

## 12. リスク・注意点

### 12-1. JNUG：金鉱株ETFは金連動ではない

最重要の誤解リスク。JNUGは「金価格に連動する」ETFではなく、「金鉱会社株（ジュニア）に連動する」ETFである。

- 金が上昇しても鉱山会社の業績・株価が低迷すれば、JNUGは下落する
- 金鉱株固有のリスク：会社倒産・生産コスト増・地政学リスク・経営判断ミス等
- ボラティリティは金現物より大幅に高く、ドラッグも大きい
- 「金価格上昇への投資」を目的とする場合は**絶対に使用してはいけない**

### 12-2. TMFの金利リスク二重化

TMFは以下の二重リスクを抱える：

1. **原資産リスク**：長期金利上昇→国債価格下落→TMF3倍下落
2. **コスト上昇リスク**：長期金利上昇→SOFR上昇→スワップコスト増大

金利が1%上昇した場合のTMF試算（イメージ）：
- 原資産（20年国債）：デュレーション約16年→金利1%上昇で約16%下落→TMFは約48%下落
- スワップコスト：SOFR+1%→実質コストも+2%増加

### 12-3. ゴールドCFDのコスト非透明性

SBI証券 金CFDはオーバーナイト金利の具体的な計算式・レートを非公開としている。

- 「カバー取引コスト反映」との記載のみで、実際の金利負担は約定後の口座明細を見るまでわからない
- 保有コストが事前に把握しづらく、長期保有計画が困難
- 他社比較が難しいため、実際に少額ポジションを数日保有して実測することを推奨

### 12-4. くりっく株365 金の廃止

- 2024年12月23日に最終決済完了。現在は利用不可
- 過去の記事・ブログ等では「くりっく株365の金がおすすめ」という情報が残っている可能性があるが、現在はサービスが存在しない

### 12-5. WisdomTree 3GOLの日本非対応

- 3倍ゴールドETFとして国際的に有名な商品だが、欧州取引所（LSE）上場のため日本の証券会社では直接購入不可
- SBI証券・楽天証券・マネックス等での取扱なし
- 「3倍金ETF」を検索すると頻繁にヒットするが、日本居住者には現状選択肢にならない

### 12-6. SOFR変動リスク

SOFR（現在3.56%）はFRBの金融政策によって変動する。利下げが進んだ場合：

- UGL（金2倍ETF）の実質コスト：4.51%→低下（SOFR連動）
- TMF（米国債3倍）の実質コスト：8.07%→低下（SOFR連動）

逆に利上げ局面では上記コストが増大する。SOFR水準によってコスト効率の商品間比較は大きく変化する。
