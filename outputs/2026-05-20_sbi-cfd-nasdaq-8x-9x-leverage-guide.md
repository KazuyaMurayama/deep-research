### 3-2. 「8倍・9倍レバレッジ」の仕組み

CFDではFXのように「レバレッジ倍率をプルダウンで選択する」という概念はない。**証拠金口座に差し入れた資金量とポジション規模の比率が実質レバレッジになる**。

```
実質レバレッジ = ポジション合計金額 ÷ 証拠金口座の残高（有効証拠金）
```

| 目標レバレッジ | 必要証拠金率（ポジション比） | 例（100万円入金・NQ100） |
|--------------|-------------------------|------------------------|
| 10倍（最大） | 10.0%（法定最低限） | ポジション1,000万円を建てる |
| **9倍** | **11.1%** | **ポジション900万円を建てる** |
| **8倍** | **12.5%** | **ポジション800万円を建てる** |
| 5倍 | 20.0% | ポジション500万円を建てる |
| 3倍 | 33.3% | ポジション300万円を建てる |

> **重要：** 法定最低証拠金率は10%（＝最大レバレッジ10倍）のため、10倍を超えることは規制上不可能。8倍・9倍はこの制約の範囲内で動かすことができる。

### 3-3. ポジション枚数計算（2026年5月時点）

SBI CFDのNQ100のポジションは「ポイント×契約倍率」で計算される。

```
【SBI CFD NQ100の取引金額計算】
NASDAQ100指数: 約29,000pt（2026年5月）
1ポイントの価値: 約1米ドル × 145円 = 約145円（1ポイント・為替レート依存）

1枚（1ロット）あたりの取引金額 = 指数値 × 1ポイント価値
例: 29,000pt × 約145円 ≒ 約420万円/枚

※実際の乗数・単位はSBI取引画面で確認が必要
```

**元本100万円での9倍レバレッジ建玉計算例：**

```
目標ポジション = 100万円 × 9倍 = 900万円
必要最低証拠金 = 900万円 × 10% = 90万円
↑ 100万円入金であれば、10万円の余裕（バッファ）が生まれる

ポジション枚数目安 = 900万円 ÷ 420万円/枚 ≒ 約2.1枚
→ 2枚建てると実質 2枚 × 420万円 = 840万円 ≒ 8.4倍相当
→ 3枚建てると実質 3枚 × 420万円 = 1,260万円（10倍超・証拠金不足）

つまり100万円の証拠金で「8〜9倍相当」は2枚建てが現実的（約8.4倍）
```

## 4. くりっく株365（NASDAQ100）での8倍・9倍レバレッジ

### 4-1. 現在のレバレッジ水準（2026年5月）

くりっく株365のNASDAQ100は東京金融取引所（TFX）が毎週証拠金基準額を更新している。

| 週 | 証拠金基準額 | NASDAQ100指数 | 実質レバレッジ |
|----|------------|--------------|--------------|
| 2026/05/18〜22 | **8,960円/枚** | 約29,000pt | **約32.4倍** |
| 2026/05/25〜29 | **8,940円/枚** | 約29,000pt | **約32.4倍** |

**計算式：**
```
取引金額（1枚）= 29,000pt × 10円/pt = 290,000円
実質レバレッジ = 290,000 ÷ 8,960 ≈ 32.4倍
```

### 4-2. くりっく株365で8倍・9倍を「実現」する方法

くりっく株365の自然なレバレッジは約32倍であり、8倍・9倍に落とすことは「意図的に余剰証拠金を大量に差し入れる」ことで対応可能。

```
【くりっく株365で9倍相当を実現する計算】
1枚あたり最低証拠金 = 8,960円
1枚あたり取引金額 = 290,000円
9倍相当の必要証拠金（1枚）= 290,000 ÷ 9 ≒ 32,222円

つまり、1枚につき32,222円の証拠金を維持することで9倍相当になる
（法定最低証拠金8,960円より大幅に多い）

10枚建ての場合：
- 最低証拠金：8,960 × 10 = 89,600円
- 9倍相当証拠金：32,222 × 10 = 322,220円
- 取引金額：290,000 × 10 = 2,900,000円（290万円）
```

> **実用上の問題点：** くりっく株365は設計上「高レバレッジ（20〜35倍）」向け商品であり、8倍・9倍への調整は「余剰資金を意図的に多く入れる」という非効率な方法になる。8倍・9倍を狙うなら**SBI CFD**の方が直感的で管理しやすい。

## 5. 具体的な操作ステップ（SBI CFDで9倍レバレッジ）

### ステップ1：SBI証券に店頭CFD口座を開設する

**前提：** SBI証券の総合口座は既に開設済みとする。

1. SBI証券にログイン
2. 上部メニュー「取引」→「CFD（株価指数・商品）」を選択
3. 「CFD口座開設」をクリック
4. 本人確認書類を用意してオンライン申込み
5. 審査期間：通常1〜3営業日
6. 審査通過後、CFD口座が有効化される

> **確認済み情報（SBI公式）：** CFD口座の開設は無料。別途、証拠金として資金を振替入金する必要がある。

### ステップ2：CFD口座へ証拠金を振替する

1. ログイン後「口座管理」→「入出金・振替」
2. SBI証券の総合口座から「CFD口座への振替」を選択
3. 振替額：目標ポジションの11.1%以上を入金（9倍レバレッジ設計なら元本100万円が必要）
4. 振替は即日反映される

**9倍レバレッジ計画の場合の入金計算：**
```
目標ポジション = 900万円（NASDAQ100で9倍）
法定最低証拠金 = 900万円 × 10% = 90万円
推奨証拠金   = 90万円 + バッファ10万円 = 100万円
（バッファを設けることでロスカットリスクを低減）
```

### ステップ3：米国NQ100銘柄で新規注文

1. 上部メニュー「CFD」→「米国株式指数CFD」または「商品一覧」
2. 「米国NQ100」を選択
3. 「新規買い」ボタンをクリック
4. 注文種別を選択（成行・指値・逆指値）
5. **ロット数（枚数）を入力**

**2026年5月時点でのロット数と実質レバレッジの目安（元本100万円）：**

| 建玉枚数 | 推定ポジション金額 | 実質レバレッジ | 必要最低証拠金 |
|---------|----------------|--------------|--------------|
| 1枚 | 約420万円 | 約4.2倍 | 約42万円 |
| **2枚** | **約840万円** | **約8.4倍** | **約84万円** |
| 2枚（小数点可） | 約900万円（2.15枚相当） | **約9.0倍** | 約90万円 |
| 3枚 | 約1,260万円 | 約12.6倍 | ⚠️証拠金不足（100万では不可） |

> **最小取引単位は0.01枚のため**、2.15枚のような細かい設定が可能。9.0倍に近づけたい場合は2.10〜2.15枚程度で調整する。

### ステップ4：証拠金維持率を管理する

注文後、SBI CFDの「証拠金維持率」を定期的に確認する。

```
証拠金維持率 = 有効証拠金 ÷ 必要証拠金 × 100（%）

SBI CFDの強制ロスカット水準：証拠金維持率100%割れ時（証拠金追加義務なし・自動決済）
証拠金維持率が100%に近づいたら：
  → 追加入金するか、一部ポジションを解消してレバレッジを下げる
```

**9倍レバレッジ（ポジション900万円、証拠金100万円）でのロスカット計算：**
```
ロスカット時の損失額 ≒ 証拠金 ≒ 100万円
これは元本100万円のほぼ全額を失うことを意味する

NASDAQ-100の下落率でのロスカット目安：
ロスカット発動下落率 ≈ 100% ÷ 9倍 ≈ 約11.1%の下落
（NASDAQ-100が11%程度下落するとロスカットラインに接触する可能性）
```

> ⚠️ **重要：** NASDAQ-100は過去に2022年だけで年間-33%下落したことがある。9倍レバレッジでは約11%の下落でロスカットの危険があるため、証拠金に十分なバッファを確保すること。

### ステップ5：金利調整額（オーバーナイトコスト）の確認

毎営業日、ポジションを翌日に持ち越すと「金利調整額」が差し引かれる。

1. SBI CFDの「ポジション一覧」→「損益詳細」で確認できる
2. 金利調整額の計算式（参考）：
   ```
   金利調整額（日次）= ポジション金額 × 年率金利 ÷ 365
   
   年率金利の目安：SOFR（3.59%）+ 業者マージン（非公開）
   参考：他社（楽天CFD）の実績はSOFR+約3%=約6.5〜7.0%/年
   SBI CFD固有の金利率は取引画面または「CFD金利調整額一覧」で確認必要
   ```
3. 年率金利が約6.5%と仮定した場合：
   ```
   900万円ポジション × 6.5% ÷ 365 ≈ 1,602円/日
   ≈ 約48,000円/月
   ≈ 約58.5万円/年
   ```

## 8. リスク・注意事項

### 8-1. レバレッジ別リスク対比表

### 8-2. 8倍・9倍レバレッジ固有のリスク

1. **急落リスク**: NASDAQ-100は高ボラティリティ指数（年率ボラ約22%）。9倍では約11%下落でロスカット到達。NASDAQ-100は週次で10%超の下落が過去に複数回発生している。

2. **オーバーナイトコストの蓄積**: 9倍レバレッジで保有すると年間コストが元本の40〜65%に達する可能性がある。「NASDAQ-100が横ばい」でも大きなコスト侵食が発生する。

3. **強制ロスカット（追証なし）**: SBI CFD（店頭CFD）は証拠金維持率が100%を下回ると**自動的に強制決済**される。投資家に「追証の支払い猶予」がない点は、ロスカット後に資金がゼロになることを意味する。

4. **証拠金維持率の監視が必須**: 特に夜間・早朝の米国市場時間帯にNASDAQ-100が急落すると、日本時間の朝に気づいたときには既にロスカットされている可能性がある。

5. **金利コストの変動**: SBI CFDの金利調整額は市場金利に連動して変動する。SOFRが上昇すれば（FRBが利上げすれば）コストが増加する。

### 8-3. 証拠金の適切なバッファ（推奨）

| 目標レバレッジ | 理論的最低証拠金（10%） | 推奨証拠金（余裕を持たせる） | 推奨レバレッジバッファ |
|--------------|----------------------|--------------------------|---------------------|
| 9倍 | 900万円の10% = 90万円 | **120万円（約7.5倍相当）** | 33%のバッファ |
| 8倍 | 800万円の10% = 80万円 | **110万円（約7.3倍相当）** | 38%のバッファ |

実際の運用では「9倍を目指すなら元本の1.2〜1.5倍の証拠金」を準備することを推奨する。

### 9-3. 最終推奨

1. **9倍よりも8倍を選択する**：11.1%下落でロスカットの9倍より、12.5%下落でロスカットの8倍の方が生存率が高い。年間コスト差は元本比5%程度であり、リスク低減効果の方が大きい。

4. **証拠金に十分なバッファを確保する**：「ギリギリ9倍」ではなく「バッファ込みで7.5〜8倍相当の証拠金」を維持することで、急落時のロスカット回避に備える。

### 10-2. 計算式まとめ

**実質レバレッジの計算：**
```
実質レバレッジ = ポジション合計金額 ÷ 有効証拠金
```

**目標レバレッジのための証拠金計算：**
```
必要証拠金 = ポジション金額 ÷ 目標レバレッジ
例：9倍・ポジション900万円 → 900万円 ÷ 9 = 100万円
```

**年間金利コスト（推計）：**
```
年間コスト = ポジション金額 × 年率金利
例：900万円 × 6.5% = 58.5万円/年
```

**ロスカット到達の下落率：**
```
ロスカット下落率 ≒ 1 ÷ レバレッジ倍率
例：9倍 → 1 ÷ 9 ≒ 11.1%の下落でロスカット危険域
```
