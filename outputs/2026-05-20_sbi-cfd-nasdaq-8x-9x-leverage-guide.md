### 3-2. 「8倍・9倍レバレッジ」の仕組み

CFDではFXのように「レバレッジ倍率をプルダウンで選択する」という概念はない。**証拠金口座に差し入れた資金量とポジション規模の比率が実質レバレッジになる**。

```
実質レバレッジ = ポジション合計金額 ÷ 証拠金口座の残高（有効証拠金）
```

| 目標レバレッジ | 必要証拠金率（ポジション比） | 例（100万円入金・NQ100） |
|--------------|-------------------------|------------------------|
| 10倍（最大） | 10.0%（法定最低限） | ポジション1,000万円を建てる |
| **9倍** | **11.1%** | **ポジション900万円を建てる** |
| **8倍** | **12.5%** | **ポジション800万円を建てる** |
| 5倍 | 20.0% | ポジション500万円を建てる |
| 3倍 | 33.3% | ポジション300万円を建てる |

> **重要：** 法定最低証拠金率は10%（＝最大レバレッジ10倍）のため、10倍を超えることは規制上不可能。8倍・9倍はこの制約の範囲内で動かすことができる。

### 3-3. ポジション枚数計算（2026年5月時点）

SBI CFDのNQ100のポジションは「ポイント×契約倍率」で計算される。

```
【SBI CFD NQ100の取引金額計算】
NASDAQ100指数: 約29,000pt（2026年5月）
1ポイントの価値: 約1米ドル × 145円 = 約145円（1ポイント・為替レート依存）

1枚（1ロット）あたりの取引金額 = 指数値 × 1ポイント価値
例: 29,000pt × 約145円 ≒ 約420万円/枚

※実際の乗数・単位はSBI取引画面で確認が必要
```

> **為替変動の影響：** 上記計算は USD/JPY = 145円を前提。1ロット単価は為替レートに比例して変動するため、145円→140円(-3.5%)では1ロット ≒ 406万円、145円→150円(+3.5%)では ≒ 435万円となる。9倍レバレッジを維持するには毎日の為替変動に合わせてロット数を調整するか、余剰証拠金を確保する必要がある。

**元本100万円での9倍レバレッジ建玉計算例：**

```
目標ポジション = 100万円 × 9倍 = 900万円
必要最低証拠金 = 900万円 × 10% = 90万円
↑ 100万円入金であれば、10万円の余裕（バッファ）が生まれる

ポジション枚数目安 = 900万円 ÷ 420万円/枚 ≒ 約2.1枚
→ 2枚建てると実質 2枚 × 420万円 = 840万円 ≒ 8.4倍相当
→ 3枚建てると実質 3枚 × 420万円 = 1,260万円（10倍超・証拠金不足）

つまり100万円の証拠金で「8〜9倍相当」は2枚建てが現実的（約8.4倍）
```

## 4. くりっく株365（NASDAQ100）での8倍・9倍レバレッジ

### 4-1. 現在のレバレッジ水準（2026年5月）

くりっく株365のNASDAQ100は東京金融取引所（TFX）が毎週証拠金基準額を更新している。

| 週 | 証拠金基準額 | NASDAQ100指数 | 実質レバレッジ |
|----|------------|--------------|--------------|
| 2026/05/18〜22 | **8,960円/枚** | 約29,000pt | **約32.4倍** |
| 2026/05/25〜29 | **8,940円/枚** | 約29,000pt | **約32.4倍** |

**計算式：**
```
取引金額（1枚）= 29,000pt × 10円/pt = 290,000円
実質レバレッジ = 290,000 ÷ 8,960 ≈ 32.4倍
```

### 4-2. くりっく株365で8倍・9倍を「実現」する方法

くりっく株365の自然なレバレッジは約32倍であり、8倍・9倍に落とすことは「意図的に余剰証拠金を大量に差し入れる」ことで対応可能。

```
【くりっく株365で9倍相当を実現する計算】
1枚あたり最低証拠金 = 8,960円
1枚あたり取引金額 = 290,000円
9倍相当の必要証拠金（1枚）= 290,000 ÷ 9 ≒ 32,222円

つまり、1枚につき32,222円の証拠金を維持することで9倍相当になる
（法定最低証拠金8,960円より大幅に多い）

10枚建ての場合：
- 最低証拠金：8,960 × 10 = 89,600円
- 9倍相当証拠金：32,222 × 10 = 322,220円
- 取引金額：290,000 × 10 = 2,900,000円（290万円）
```

> **実用上の問題点：** くりっく株365は設計上「高レバレッジ（20〜35倍）」向け商品であり、8倍・9倍への調整は「余剰資金を意図的に多く入れる」という非効率な方法になる。8倍・9倍を狙うなら**SBI CFD**の方が直感的で管理しやすい。

## 5. 具体的な操作ステップ（SBI CFDで9倍レバレッジ）

### ステップ1：SBI証券に店頭CFD口座を開設する

**前提：** SBI証券の総合口座は既に開設済みとする。

1. SBI証券にログイン
2. 上部メニュー「取引」→「CFD（株価指数・商品）」を選択
3. 「CFD口座開設」をクリック
4. 本人確認書類を用意してオンライン申込み
5. 審査期間：通常1〜3営業日
6. 審査通過後、CFD口座が有効化される

> **確認済み情報（SBI公式）：** CFD口座の開設は無料。別途、証拠金として資金を振替入金する必要がある。

### ステップ2：CFD口座へ証拠金を振替する

1. ログイン後「口座管理」→「入出金・振替」
2. SBI証券の総合口座から「CFD口座への振替」を選択
3. 振替額：法定最低限は「ポジション金額の10%以上」だが、現実的な耐久力のために大幅なバッファが必要
4. 振替は即日反映される

**9倍レバレッジ計画の場合の入金計算：**
```
目標ポジション = 900万円（NASDAQ100で9倍）
法定最低証拠金 = 900万円 × 10% = 90万円

【入金額別のロスカット耐久下落率】
- 100万円入金時：バッファ10万円 → 約1.2%の下落でロスカット発動（危険）
- 200万円入金時：バッファ110万円 → 約13.6%の下落でロスカット発動（現実的）
- ※200万円入金の場合の実効レバレッジ = 900÷200 = 4.5倍

推奨入金：200万円以上（ポジション900万円を維持する場合）
※ただし200万円でも約14%下落でロスカット。2022年には1四半期で-25%超の局面があった点に注意。
```

### ステップ3：米国NQ100銘柄で新規注文

1. 上部メニュー「CFD」→「米国株式指数CFD」または「商品一覧」
2. 「米国NQ100」を選択
3. 「新規買い」ボタンをクリック
4. 注文種別を選択（成行・指値・逆指値）
5. **ロット数（枚数）を入力**

**2026年5月時点でのロット数と実質レバレッジの目安（元本100万円）：**

| 建玉枚数 | 推定ポジション金額 | 実質レバレッジ | 必要最低証拠金 |
|---------|----------------|--------------|--------------|
| 1枚 | 約420万円 | 約4.2倍 | 約42万円 |
| **2枚** | **約840万円** | **約8.4倍** | **約84万円** |
| 2枚（小数点可） | 約900万円（2.15枚相当） | **約9.0倍** | 約90万円 |
| 3枚 | 約1,260万円 | 約12.6倍 | ⚠️証拠金不足（100万では不可） |

> **最小取引単位は0.01枚のため**、2.15枚のような細かい設定が可能。9.0倍に近づけたい場合は2.10〜2.15枚程度で調整する。

### ステップ4：証拠金維持率を管理する

注文後、SBI CFDの「証拠金維持率」を定期的に確認する。

```
証拠金維持率 = 有効証拠金 ÷ 必要証拠金 × 100（%）

SBI CFDの強制ロスカット水準：証拠金維持率100%割れ時（証拠金追加義務なし・自動決済）
証拠金維持率が100%に近づいたら：
  → 追加入金するか、一部ポジションを解消してレバレッジを下げる
```

**ロスカット発動の正しい計算式（重要）：**

ロスカット下落率 = （有効証拠金 − 必要最低証拠金） ÷ ポジション金額 × 100
               = 証拠金バッファ ÷ ポジション金額

CFDのロスカットは「証拠金維持率100%割れ（有効証拠金＝必要最低証拠金）」で発動する。
全資本が消えた時点ではなく、バッファが尽きた時点でロスカットされる点に注意。

**証拠金別・ロスカット発動下落率の試算（9倍・ポジション900万円）：**

| 入金額 | 実効レバレッジ | ロスカット下落率（精密式） |
|-------|------------|----------------------|
| 100万円（ギリギリ）| 9.0倍 | **約1.2%下落** |
| 150万円 | 6.0倍 | **約7.4%下落** |
| **200万円（推奨）** | **4.5倍** | **約13.6%下落** |
| 270万円（安全） | 3.3倍 | **約22.2%下落** |

精密式: X = (M − 0.1×P) / (0.9×P)、ポジションP=900万円固定。

⚠️ **100万円を入金して900万円のポジションを建てると、NASDAQ-100がわずか1%下落しただけでロスカット危機に陥る。「9倍レバレッジを11%の下落まで耐えられる」というのは誤りであり、現実には200万円以上の証拠金が必要。**

証拠金維持率が150%以下になったら追加入金またはポジション縮小を検討：
150%維持率達成には：必要証拠金 × 1.5 ≒ 135万円が必要。
100万円入金時の初期維持率は100/90 × 100 ≒ 111%（150%を下回っている）。

> ⚠️ **重要：** NASDAQ-100は過去に2022年だけで年間-33%下落したことがある。9倍レバレッジで現実的な耐久力（10〜15%の下落に対応）を持たせるには、**200万円以上の証拠金**が必要。

### ステップ5：金利調整額（オーバーナイトコスト）の確認

毎営業日、ポジションを翌日に持ち越すと「金利調整額」が差し引かれる。

1. SBI CFDの「ポジション一覧」→「損益詳細」で確認できる
2. 金利調整額の計算式（参考）：
   ```
   金利調整額（日次）= ポジション金額 × 年率金利 ÷ 365
   
   年率金利の目安：SOFR（3.59%）+ 業者マージン（非公開）
   参考：他社（楽天CFD）の実績はSOFR+約3%=約6.5〜7.0%/年
   SBI CFD固有の金利率は取引画面または「CFD金利調整額一覧」で確認必要
   ```
3. 年率金利が約6.5%と仮定した場合：
   ```
   900万円ポジション × 6.5% ÷ 365 ≈ 1,602円/日
   ≈ 約48,000円/月
   ≈ 約58.5万円/年
   ```

## 8. リスク・注意事項

### 8-1. レバレッジ別リスク対比表

**【重要】ロスカット下落率はレバレッジと入金方法により大きく異なる。以下2ケースを区別して把握すること。**

**ケースA：同じレバレッジを維持したまま入金を倍増する場合（ポジションも比例増）**

同じレバレッジを維持しながら入金を増やしても、ポジションも比例増するためロスカット下落率は変わらない。

| 目標レバレッジ | 入金100万円・ポジション | 入金200万円・ポジション | ロスカット下落率（一定） |
|--------------|---------------------|---------------------|----------------------|
| 5倍 | 100万円・500万円 | 200万円・1,000万円 | **約11.1%**（不変） |
| 8倍 | 100万円・800万円 | 200万円・1,600万円 | **約2.8%**（不変） |
| 9倍 | 100万円・900万円 | 200万円・1,800万円 | **約1.2%**（不変） |
| 10倍 | 100万円・1,000万円 | 200万円・2,000万円 | **即時**（不変） |

> ロスカット下落率 X = (M − 0.1P) / (0.9P)。レバレッジを維持する限り入金増加は生存率を上げない。

**ケースB：ポジションを900万円に固定したまま入金額を変える場合（実効レバレッジが低下）**

| 入金額 | ポジション | 実効レバレッジ | ロスカット下落率 |
|-------|----------|-------------|----------------|
| 100万円 | 900万円 | 9.0倍 | **約1.2%** |
| 150万円 | 900万円 | 6.0倍 | **約7.4%** |
| **200万円（推奨）** | **900万円** | **4.5倍** | **約13.6%** |
| 270万円 | 900万円 | 3.3倍 | **約22.2%** |

> 「9倍レバレッジを安全に運用したい」なら、入金を増やすとポジション実効レバレッジが低下する。200万円入金・900万円ポジション（実効4.5倍）で約13.6%の耐久力。

⚠️ **NASDAQ-100は2022年の年間下落-33%、コロナショック時に1日-10%超の急落がある。1.2%（9倍レバレッジの最低バッファ）は数時間でヒットする可能性がある。**

### 8-2. 8倍・9倍レバレッジ固有のリスク

1. **急落リスク**: NASDAQ-100は高ボラティリティ指数（年率ボラ約22%）。**9倍レバレッジを100万円入金で運用すると、わずか1.1%の下落でロスカット危機**。200万円入金でも約12%の下落で強制決済される。NASDAQ-100は週次で5%超の下落が頻発しており、100万円入金での9倍運用は事実上デイトレードか、入金翌日には証拠金が消滅するリスクがある。

2. **オーバーナイトコストの蓄積**: 9倍レバレッジで保有すると年間コストが元本の**45〜65%**に達する可能性がある。「NASDAQ-100が横ばい」でも大きなコスト侵食が発生する。

3. **強制ロスカット（追証なし）**: SBI CFD（店頭CFD）は証拠金維持率が100%を下回ると**自動的に強制決済**される。投資家に「追証の支払い猶予」がない点は、ロスカット後に資金がゼロになることを意味する。

4. **証拠金維持率の監視が必須**: 特に夜間・早朝の米国市場時間帯にNASDAQ-100が急落すると、日本時間の朝に気づいたときには既にロスカットされている可能性がある。

5. **金利コストの変動**: SBI CFDの金利調整額は市場金利に連動して変動する。SOFRが上昇すれば（FRBが利上げすれば）コストが増加する。

### 8-3. 証拠金の適切なバッファ（推奨）

| 初期目標 | ポジション | 最低証拠金（法定） | 推奨入金（安全運用） | 推奨時の実効レバレッジ | 推奨時のロスカット耐久 |
|---------|---------|----------------|------------------|--------------------|---------------------|
| 9倍相当 | 900万円 | 90万円 | **200万円** | 4.5倍 | **約13.6%** |
| 8倍相当 | 800万円 | 80万円 | **180万円** | 4.4倍 | **約14.0%** |

精密式: 9倍・200万円時 X = (200-90)/(0.9×900) = 110/810 = 13.6% / 8倍・180万円時 X = (180-80)/(0.9×800) = 100/720 = 13.9%≒14.0%

> **補足：120万円（9倍時）や110万円（8倍時）ではロスカット耐久は3〜4%にとどまる。NASDAQ-100は日次で2〜5%下落することがあるため、実務的には200万円以上の証拠金が必要。「元本の1.2〜1.5倍の証拠金」という従来の推奨は過小評価であり、「元本の2倍以上の証拠金を確保したうえでポジションを取る」ことを強く推奨する。**

5. 毎日の金利調整額と証拠金維持率を確認

### 9-3. 最終推奨

NASDAQ-100に8倍・9倍のレバレッジをSBI証券でかけることは**技術的・法的に完全に可能**。

ただし、以下の点を強く推奨する：

1. **9倍よりも8倍を選択する**：同じ100万円入金時のロスカット下落率は9倍=1.23%、8倍=2.78%（精密式）。200万円入金・ポジション固定時は9倍=13.6%耐久、8倍=16.7%耐久となり、8倍の方が安全性が高い。

2. **金利コストを先に把握してから取引開始する**：SBI CFD固有の年率金利は非公開のため、取引画面でログイン後に確認してから計画を立てること。

3. **長期保有には向かない**：9倍レバレッジで年間コストが元本の50〜60%を超える場合、NASDAQ-100が同等以上のリターンを上げないとマイナスになる。短中期のトレンドフォロー戦略に限定することが賢明。

4. **証拠金に十分なバッファを確保する**：「ギリギリ9倍」ではなく入金を増やしてポジション固定のまま実効レバレッジを4〜5倍程度に収めることで、急落時のロスカット回避に備える。

5. **長期保有ではKelly基準を超えた過剰レバレッジ：** NASDAQ-100の期待リターン約10%、年率ボラ約20%、無リスク金利4%でKelly最適レバレッジを算定すると f* ≈ (10%−4%)/20%² ≈ 1.5倍。9倍はKelly最適の約6倍であり、理論上長期保有では破産確率が極めて高い。8〜9倍レバレッジは**明確な相場観に基づく短期スイング（数日〜数週間）に限定**し、長期インデックス投資への転用は推奨しない。

### 10-2. 計算式まとめ

**実質レバレッジの計算：**
```
実質レバレッジ = ポジション合計金額 ÷ 有効証拠金
```

**目標レバレッジのための証拠金計算：**
```
必要証拠金 = ポジション金額 ÷ 目標レバレッジ
例：9倍・ポジション900万円 → 900万円 ÷ 9 = 100万円
```

**年間金利コスト（推計）：**
```
年間コスト = ポジション金額 × 年率金利
例：900万円 × 6.5% = 58.5万円/年
```

**ロスカット発動の下落率（精密計算式）：**
```
精密式: X = (M − r × P) / ((1 − r) × P)
        = (有効証拠金 − 必要最低証拠金) / (必要証拠金の残額ウェイト)

M = 有効証拠金（入金額）, P = ポジション金額, r = 証拠金率 = 0.10

※簡易近似式 X ≈ (M − rP) / P は精密式の約90%の値となり、わずかに保守的（低め）な推計。
※本レポートでは精密式の値を使用。

例1：入金100万円・9倍（P=900万円）
   → X = (100 − 0.1×900) / (0.9×900) = 10/810 ≒ 1.23%の下落でロスカット危険域

例2：入金200万円・P=900万円（実効4.5倍）
   → X = (200 − 0.1×900) / (0.9×900) = 110/810 ≒ 13.58%の下落でロスカット危険域
   ※この場合のポジションは900万円のまま（実効レバレッジ4.5倍に低下）

例3：入金200万円・9倍維持（P=1,800万円）
   → X = (200 − 0.1×1,800) / (0.9×1,800) = (200−180)/1,620 = 20/1,620 ≒ 1.23%
   ※9倍を維持してポジションも増やすと、ロスカット率は100万円入金時と変わらない
```
