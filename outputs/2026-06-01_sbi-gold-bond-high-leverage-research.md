## 4. 手数料・金利コストの詳細算出

**共通前提（2026年6月時点）：**
- SOFR = **3.59%**（2026年6月の参照値、FRED/NY Fed系データに基づく [推計]）
- 金の年率ボラティリティ σ = 16.2%
- 長期米国債（TLT）の年率ボラティリティ σ = 12.5%
- 元本＝計算基準。為替リスクは別途。

### 4-1. CFD系のコスト計算（SBI金CFD）

CFDは日次リバランスをしないため**ボラドラッグはゼロ**ですが、ポジションを翌日に持ち越すと「オーバーナイト金利」がかかります。

**計算式：**
```
オーバーナイト金利（年率・推計） = SOFR + 金リース金利 + 業者マージン
                              = 3.59% + 0.3%[推計] + 0.5〜1.0%[推計]
                              = 約4.4〜4.9%/年 [推計]

スプレッドコスト = スプレッド幅 × 1ポイント価値 × 枚数（往復1回のみ）

合計保有コスト = オーバーナイト金利 × ポジション金額 × 保有日数/365 + スプレッドコスト
```

**重要：レバレッジ倍率とコストの関係**
- CFDのオーバーナイト金利は「**実際に建てたポジションの想定元本（名目金額）**」にかかります。
- レバレッジ5倍で元本100万円を投じると、想定元本は**500万円**。金利は500万円に対してかかります。

**例：元本100万円・レバレッジ5倍（想定元本500万円）・金利4.7%[推計]の場合**
```
1ヶ月（30日）：500万 × 4.7% × 30/365  ≈ 19,300円
3ヶ月（90日）：500万 × 4.7% × 90/365  ≈ 57,900円
1年（365日） ：500万 × 4.7% × 365/365 ≈ 235,000円
```
→ 元本100万円に対し、**1年保有でコストだけで約23.5%（＝23.5万円）**。これに往復スプレッドが加わります。

> ⚠️ SBI証券は金CFDのオーバーナイト金利の具体的計算式・レートを**非公開**としています（2026年5月時点で金・銀のスプレッドも各社拡大中・非開示）。上記は業界標準に基づく推計値で、実額は口座明細での実測が必要です。

### 4-2. ETF系のコスト計算（TMF・5倍ETP等）

日次リバランス型ETF/ETPは、内部のスワップ契約に**ファイナンスコスト（SOFR連動）**が埋め込まれ、さらに**ボラドラッグ**が発生します。

**計算式：**
```
スワップ埋込コスト = SOFR × (レバレッジ - 1)
実質年率コスト    = 経費率 + スワップ埋込コスト
ボラドラッグ      = 0.5 × L × (L-1) × σ²
合計コスト目安    = 実質年率コスト + ボラドラッグ
```

**各レバレッジ倍率別の年率コスト試算（SOFR=3.59%）**

| 商品 | L | 経費率 | スワップ埋込＝SOFR×(L-1) | ボラドラッグ＝0.5×L×(L-1)×σ² | 合計年率コスト目安 |
|------|---|--------|-------------------------|------------------------------|-------------------|
| UGL（金2倍） | 2 | 0.95% | 3.59%×1 = 3.59% | 0.5×2×1×0.162² = 2.62% | **約7.2%/年** [推計] |
| 3GOL（金3倍） | 3 | 0.99% | 3.59%×2 = 7.18% | 0.5×3×2×0.162² = 7.87% | **約16.0%/年** [推計] |
| **GLD5（金5倍）** | 5 | 0.75%[推計] | 3.59%×4 = 14.36% | 0.5×5×4×0.162² = 26.2% | **約41.3%/年** [推計] |
| TMF（債券3倍） | 3 | 0.95% | 3.59%×2 = 7.18% | 0.5×3×2×0.125² = 4.69% | **約12.8%/年** [推計] |
| **TLT5（債券5倍）** | 5 | 0.75%[推計] | 3.59%×4 = 14.36% | 0.5×5×4×0.125² = 15.6% | **約30.7%/年** [推計] |

> ⚠️ **5倍ETPのボラドラッグは破壊的**。金5倍（GLD5）は横ばい相場でも年率40%超のコスト相当が発生する計算で、**長期保有は事実上不可能**。これらは「数日〜数週間の短期トレード専用ツール」と理解すべきです。

## 5. 実際の運用例シミュレーション

### 5-1. シナリオA（ゴールドCFD強気局面）

**前提：** 元本200万円、SBI金CFD、レバレッジ5倍、金価格が3ヶ月で+15%上昇

```
想定元本（名目ポジション） = 200万円 × 5倍 = 1,000万円
金価格上昇による損益 = 1,000万円 × (+15%) = +150万円（粗利益）

オーバーナイト金利コスト（金利4.7%[推計]・90日）
  = 1,000万円 × 4.7% × 90/365 ≈ -115,800円

スプレッドコスト（往復・概算） ≈ -数千〜数万円 [要確認・非開示]

差引利益（税引前） ≈ +150万円 - 約11.6万円 ≈ 約138万円
元本200万円に対するリターン ≈ +69%（3ヶ月） [推計]
税引後（申告分離20.315%）≈ +138万円 ×(1-0.20315) ≈ 約110万円
```

**ポイント：** 5倍なら金+15%が元本ベースで+69%に増幅。ただし**逆に金が-15%なら元本ベース-75%＋金利**で、ロスカット必至。

### 5-2. シナリオB（TMF・利下げ局面）

**前提：** 元本100万円、TMF（3倍）、長期金利が1%低下、保有6ヶ月

```
ステップ1：原資産（20年超国債/TLT）の値動き
  TLTのデュレーション ≈ 16年
  金利1%低下 → 債券価格 ≈ +16%上昇

ステップ2：TMFは日次3倍 → 単純計算で +48%
  （ただし日次リバランスのため、滑らかな1%低下なら概ね近い。
   往復のある相場ではボラドラッグで目減り）

ステップ3：6ヶ月のコスト差引
  実質年率コスト＋ボラドラッグ ≈ 12.8%/年[推計]
  6ヶ月分 = 12.8% × 0.5 ≈ 6.4%

ステップ4：実質損益
  粗リターン +48% - コスト6.4% ≈ +41.6%（税引前） [推計]
  100万円 → 約141.6万円
  税引後（20.315%）≈ +41.6万 ×(1-0.20315) ≈ +33.2万円 → 約133万円
```

**ポイント：** 利下げ1%でTMFは元本+40%超のポテンシャル。**利下げ局面はTMFの最も得意な相場**。ただし金利が逆に上がれば3倍下落＋コストの二重苦。

### 5-3. シナリオC（高レバレッジ急落失敗例）

**前提：** 元本100万円、9倍レバレッジ（参考・CFDで任意設定した想定）、市場が急落

```
想定元本 = 100万円 × 9倍 = 900万円
ロスカット水準の目安：証拠金維持率が一定割合（例：証拠金の50%）を割ると強制決済

【金価格が短期間で -8% 下落した場合】
  損益 = 900万円 × (-8%) = -72万円
  元本100万円に対する損失率 = -72%
  → 証拠金100万円のうち72万円を失い、維持率が急落
  → 多くの業者で-8〜-11%の逆行でロスカット発動

【金価格が -11% 下落した場合】
  損益 = 900万円 × (-11%) = -99万円
  → 元本100万円がほぼ全損。ロスカットが間に合わなければ追証（不足金）も発生し得る
```

**ロスカットまでの逆行幅の目安（証拠金100%消費時）：**

| レバレッジ | 全損に必要な逆行幅 | ロスカット目安（維持率50%割れ） |
|-----------|-------------------|--------------------------------|
| 3倍 | -33% | -16〜-22%程度 |
| 5倍 | -20% | -10〜-13%程度 |
| 9倍 | -11% | -5〜-7%程度 |
| 20倍 | -5% | -2〜-3%程度 |

> ⚠️ 9倍では金がわずか-5〜7%動くだけでロスカット。金は1日で2〜3%動くことも珍しくないため、**9倍以上は数日の保有でも危険**。必ず損切りラインを事前設定し、想定元本に対する1日の値動き（σ÷√250 ≈ 日次1%程度）を意識すること。

**読み方：** 右端列は「3倍レバレッジETFの内部スワップコスト＝金利×(3-1)＝金利×2」の目安です。**金利が高い時代ほど、3倍ETFを持つだけで失うコストが大きい**ことが一目でわかります。

### 6-2. 金利環境別のTMF実質コスト試算

TMF（債券3倍）の「経費率0.95%＋スワップ埋込（金利×2）」での実質年率コスト [推計]：

| 金利環境（FFレート目安） | 代表的な時期 | スワップ埋込(×2) | TMF実質年率コスト(経費率込) |
|--------------------------|--------------|------------------|-----------------------------|
| 0.1%（ゼロ金利） | 2008-2015, 2020-2021 | 0.2% | **約1.15%/年** |
| 1.5%（正常化初期） | 2016-2018 | 3.0% | 約3.95%/年 |
| 3.6%（現在・2026） | 2026年6月 | 7.18% | **約8.13%/年** |
| 5.25%（利上げピーク） | 2006, 2023 | 10.5% | **約11.45%/年** |
| 19%（Volcker期） | 1981 | 38% | 約38.95%/年（仮にETFが存在したら） |

**示唆：** TMF/TLT5のようなレバレッジ債券ETFは、**ゼロ金利時代（2020-2021）には保有コストがほぼ無料**でした。当時TMFが人気化したのはこのためです。しかし現在（金利3.6%）はコストが約8%/年に跳ね上がっており、**同じ商品でも金利環境次第で「ほぼ無料」から「重荷」へ激変**します。

### 6-3. 現在の金利水準と今後の見通し

- **SOFR：約3.59%**（2026年6月時点の参照値）。FFレート誘導目標は **3.50〜3.75%**（2025年12月時点）で、2024年9月以降に累計約1.75%pt引き下げられました。
- 市場予想（Long Forecast系）では2026年6月のSOFRは概ね3.5%前後で推移し、緩やかな低下基調 [推計]。
- **レバレッジ商品への含意：** 利下げが進めばスワップ埋込コストが下がり、TMF・5倍ETP・金CFDのオーバーナイト金利すべてが軽くなります。**「利下げ局面」は債券レバレッジ（TMF/TLT5）にとって、値上がり益とコスト低下のダブルメリット**となる稀有な好機です。逆に利上げ再開なら全レバレッジ商品のコストが悪化します。

## 8. リスク・注意事項

1. **5倍ETP（GLD5・TLT5）はリテール非対応**：発行体Leverage Sharesは「リテール投資家に直接販売しない」と明記。日本のネット証券では買えず、IG証券等のCFD経由になるが、当該銘柄の取扱は**口座開設後に要確認**。
2. **5倍ETPの長期保有は構造的に破綻**：金5倍ETP（GLD5）は横ばいでも年率40%超のコスト相当。**数日〜数週間の短期専用**。
3. **CFDのコスト非透明性**：SBI金CFDはオーバーナイト金利・スプレッドを非開示（2026年5月時点で金・銀スプレッドは各社拡大中）。実額は口座明細での実測が必須。
4. **高倍率のロスカットリスク**：9倍で-5〜7%、20倍で-2〜3%の逆行でロスカット。金は1日2〜3%動くため、超高倍率の保有は数日でも危険。必ず損切りライン設定を。
5. **金利上昇でコスト悪化**：すべてのレバレッジ商品はSOFR連動。利上げ再開なら保有コストが膨らむ。
6. **為替リスク**：欧州ETP・米国ETF・CFDはドル／ポンド建て。円高で円換算リターンが目減りする。
7. **税制**：CFD・先物は「先物取引に係る雑所得等（申告分離20.315%）」、米国ETF（TMF等）は「株式等（申告分離20.315%）」。両者は損益通算不可。NISA対象外。
8. **「6倍・10倍のゴールド／ボンドETF」は未確認 [要確認]**：商品・債券分野では5倍が市販ETP上限。6倍超はCFDの任意倍率設定でのみ実現可能。
