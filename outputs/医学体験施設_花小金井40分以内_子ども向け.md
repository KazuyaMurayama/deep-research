# 子ども向け医学体験施設リサーチ
## テーマ：医学を学ぶ・体験できる施設（花小金井から40分以内）

**作成者：男座員也（Kazuya Oza）**  
**作成日：2026-05-04**

---

## 概要

花小金井駅（西武新宿線・東京都小平市）を起点に、電車・バス40分以内で行ける「子どもが医学を学べる・体験できる施設」を調査した。

---

## 施設一覧

### 1. 多摩六都科学館 ⭐ 最短アクセス

| 項目 | 内容 |
|------|------|
| **住所** | 東京都西東京市芝久保町5-10-64 |
| **花小金井からのアクセス** | 花小金井駅北口 → はなバス第4北ルート（田無駅行）約8分・150円 |
| **所要時間** | 約8分 |
| **料金（子ども）** | 展示のみ210円 / 展示＋プラネタリウム420円 / フルセット530円 |
| **開館時間** | 9:30〜17:00（発券16:00まで） |
| **休館日** | 火曜日 |

**医学・人体関連の内容：**
- **「からだの部屋」**：五感・人体の仕組みを実際に触れて体験できる常設展示室
- 明治薬科大学セルフメディケーション学研究室と連携した健康づくり展示あり
- 視覚・嗅覚・聴覚など五感の仕組みを体験するコーナーを設置

**こんな子に向いている：**
小学生低学年から楽しめる。費用が低く、気軽に立ち寄れる。プラネタリウムと組み合わせた半日コースも可能。

[📍 Googleマップで検索](https://www.google.com/maps/search/多摩六都科学館+西東京市) ｜ [🌐 公式サイト](https://www.tamarokuto.or.jp/) ｜ [🚌 はなバス時刻表](https://www.tamarokuto.or.jp/hanabus/)

### 2. Gifte! 本格キッズドクター体験（外科）@ 東京外科クリニック

| 項目 | 内容 |
|------|------|
| **住所** | 東京都千代田区神田小川町 |
| **最寄り駅** | 都営小川町駅・東京メトロ淡路町駅 B5出口 徒歩約3分 |
| **花小金井からのアクセス** | 西武新宿線→都営新宿線乗換 約39分 |
| **所要時間** | 約39分 |
| **対象年齢** | 小学4〜6年生 |
| **料金** | 15,000円/組（子ども1人＋保護者1人） |
| **形式** | 完全予約制・少人数制 |

**体験内容：**
- 医師の白衣を着用し、日帰り内視鏡手術が行われる本格クリニックを見学
- バックヤードツアー：本物の内視鏡設備の見学
- **電気メス**の操作を実体験
- プロの役者が「患者役」を演じ、子どもが診断・治療方針を選択
- 外科医の仕事を臨場感を持って学べる

**こんな子に向いている：**
外科・手術に強い興味を持つ小学校高学年向け。費用はかかるが、本物の医療機器を使った非常に本格的な体験ができる。

[📍 Googleマップで検索](https://www.google.com/maps/search/東京外科クリニック+千代田区神田小川町) ｜ [🌐 Gifte公式](https://www.gifte.jp/7010K000000NGgF)

### 3. Gifte! キッズドクター体験（循環器内科）@ つかさ内科

| 項目 | 内容 |
|------|------|
| **住所** | 東京都板橋区 |
| **最寄り駅** | 都営三田線 本蓮沼駅 徒歩約7分 |
| **花小金井からのアクセス** | 西武新宿線→山手線→三田線乗換 約38〜42分 |
| **所要時間** | 約38〜42分（時間帯により変動） |
| **対象年齢** | 小学1〜6年生（推奨：3〜6年生） |
| **料金** | 18,700円/組（子ども1人＋保護者1人） |
| **形式** | 完全予約制・1回20組定員 |

**体験内容（4種類）：**
1. **頸動脈エコー診察体験**：超音波装置で血管を観察
2. **レントゲン診察体験**：X線画像の見方を学ぶ
3. **聴診ドクター体験**：聴診器で心音・呼吸音を確認
4. **リハビリテーション体験**：理学療法士の仕事を体験

内科医・医学博士が監修し、「長寿の秘密」というテーマで循環器の仕組みを楽しく学べる。

**こんな子に向いている：**
体の中の仕組みに興味がある子。低学年でも参加可能で、内科系の医療体験として充実している。40分の上限ギリギリなのでアクセス時間を要確認。

[📍 Googleマップで検索](https://www.google.com/maps/search/つかさ内科+板橋区+本蓮沼) ｜ [🌐 Gifte公式](https://www.gifte.jp/7010K000001uupY) ｜ [📖 いこーよ掲載](https://iko-yo.net/events/505047)

### 4. こども科学センター・ハチラボ

| 項目 | 内容 |
|------|------|
| **住所** | 東京都渋谷区桜丘町23-21（渋谷区文化総合センター大和田 内） |
| **最寄り駅** | JR渋谷駅 徒歩約5分 |
| **花小金井からのアクセス** | 西武新宿線→JR山手線 乗換1回 約37分 |
| **所要時間** | 約37分 |
| **料金** | 無料 |
| **対象年齢** | 主に小中学生 |

**内容：**
- 渋谷区立の体験型科学センター（入場無料）
- 年5回テーマが変わる特別展示（「自然・技術・数学・環境SDGs」など）
- 土日・長期休暇中心にワークショップ・実験・工作を開催
- 健康・身体に関する特別展も不定期で開催

**注意：** 常設で医療特化の展示があるわけではなく、展示テーマは時期によって異なる。訪問前に公式サイトで展示内容を確認推奨。

**こんな子に向いている：**
費用をかけずに科学全般を体験したい場合に最適。医療系の特別展や、身体・生命に関する展示が行われる時期を狙うとより効果的。

[📍 Googleマップで検索](https://www.google.com/maps/search/こども科学センター+ハチラボ+渋谷) ｜ [🌐 公式サイト](https://shibu-cul.jp/hachilabo)

## 推薦まとめ

### 手軽に行くなら → **多摩六都科学館**
- 花小金井から最も近い（8分）
- 入場料が安く気軽
- 「からだの部屋」で人体の仕組みを楽しく体験
- プラネタリウムとセットで半日楽しめる

### 本格的に医療を体験したいなら → **Gifte! キッズドクター体験**
- 外科体験（東京外科クリニック）：電気メス・内視鏡などリアルな手術環境
- 内科体験（つかさ内科）：エコー・レントゲン・聴診の本格体験
- 費用はかかるが「本物の医療機器を使う」唯一の体験
- 要事前予約（Gifte公式サイトから）
