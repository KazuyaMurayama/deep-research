## 1. 法改正の概要

### 2026年4月1日施行：自転車への交通反則通告制度（青切符）

2026年4月1日より、**16歳以上の自転車運転者**を対象に「交通反則通告制度（青切符）」が導入された。

| 項目 | 内容 |
|------|------|
| 対象年齢 | 16歳以上 |
| 反則金水準 | 5,000円〜12,000円（違反内容による） |
| 酒気帯び運転 | 赤切符（刑事罰）の対象 |
| 手続き | 7日以内に仮納付で刑事手続き回避 |

**対象となる主な違反行為（反則金）：**

| 違反種別 | 反則金 |
|---------|--------|
| 信号無視 | 6,000円 |
| 指定場所一時不停止 | 5,000円 |
| 携帯電話使用（ながらスマホ） | 12,000円 |
| 通行区分違反 | 5,000円 |
| 歩道通行方法違反 | 5,000円 |
| 傘差し運転・制動装置不良 | 5,000円 |

### 2026年5月23日施行：車道走行の間隔規定

自動車と自転車が同一車道を走行する際のルールが明文化・強化された。

| 規定対象 | 内容 |
|---------|------|
| 自動車側 | 自転車追い越し時、**1.5m以上の間隔確保**（推奨）または徐行義務 |
| 自動車の罰則 | 反則金7,000円（普通車）、2点減点；悪質な場合は3ヶ月以下の拘禁刑または5万円以下の罰金 |
| 自転車側 | **できる限り道路の左端に寄って走行する義務** |
| 自転車の罰則 | 5万円以下の罰金 |

> ⚠️ 1.5mルールは2026年5月23日施行のため、本レポートの集計期間（5月末まで）への影響は**約1週間程度**に限定される。

## 2. 昨年同時期との比較（2026年 vs 2025年）

### 東京都（警視庁）データ：2026年1〜6月初旬 vs 2025年全年

> **注：** 2026年の全国月別データは未公表（警察庁の月報は通常翌月中旬以降に公開）。東京都データを最良推計として使用する。

| 指標 | 2025年（東京・年間） | 2026年（東京・1〜6月3日YTD） | 参考：2024年（東京・年間） |
|------|---------------------|------------------------------|--------------------------|
| 自転車事故件数 | 15,109件 | — | 15,080件 |
| 自転車乗用中死者数 | 21人 | **6人** | 25人 |
| 自転車乗用中負傷者数 | 12,961人 | — | 12,903人 |

**前年比（月別ペース換算）：**

- 2025年の月平均死者数 = 21人 ÷ 12 = **1.75人/月**
- 2026年の1〜5月（5ヶ月）の月平均 = 6人 ÷ 5 = **1.2人/月**
- 推計前年同期比：**▲31%（減少）**

> この減少が青切符制度等の法改正効果なのか、季節・天候要因なのかは、年間通じた追跡が必要。

## 5. 法改正の効果

### 評価サマリー

| 改正内容 | 施行日 | 初期評価 | エビデンス |
|---------|--------|---------|---------|
| 青切符制度 | 2026年4月1日 | △（取締件数は想定の半分） | 取締2,147件/月（月平均4,268件比▲49.7%） |
| 追い越し間隔規定 | 2026年5月23日 | 評価困難 | 施行直後（約10日間のデータのみ） |
| ヘルメット努力義務 | 2023年4月1日 | ◯（死亡者数減少に貢献） | 着用率21.2%（低いが上昇傾向） |

### 青切符制度の現状と課題

```
【第1ヶ月（2026年4月）の取締り状況】

青切符告知: 2,147件
  ├─ 一時不停止: 846件（40%）
  ├─ ながらスマホ: 713件（33%）
  └─ 信号無視: 298件（14%）

指導警告: 135,855件
  └─ 法令・制度未認知と思われる軽微違反

→ 認知・啓発の遅れが主要課題
→ 2024年時点での対象違反月平均4,268件に対し、約半分の水準
```

### 中長期的な改正効果への期待

1. **青切符の浸透効果**：制度の認知が広まるにつれ、自転車の交通ルール遵守率が向上することが期待される
2. **追い越し間隔規定**：自動車側の意識改革による自転車との接触事故減少が見込まれる
3. **ヘルメット着用促進**：努力義務化（2023年）以降、死亡者数は減少傾向。着用率21.2%はまだ低く、更なる向上余地がある

## 6. ヘルメット着用と死亡リスク

| 指標 | 数値 | 出典 |
|------|------|------|
| ヘルメット着用率（2026年6月・全国） | **21.2%** | 警察庁調査 |
| 自転車死亡事故のうち頭部致命傷の割合 | 約**70%** | 警察庁 |
| ヘルメット非着用の致死率（着用比） | **約2.3倍** | 警察庁統計 |
| 5年間（2019〜2023年）の致死率比 | **約1.9倍** | 警察庁分析 |
| 死亡者のうちヘルメット非着用の割合（2023年） | **約90%超** | 警察庁 |

> **含意：** ヘルメット着用が強制義務化されれば、自転車死亡者数を最大50〜60%削減できる可能性がある（致死率2.3倍の逆数から推計）。

## 7. 総括と留意事項

### エグゼクティブサマリー

1. **死者数はピーク（2023年：346人）から減少傾向**
   - 2023 → 2024: ▲5.5%（327人）
   - 2024 → 2025: 推計▲4.3%（約313人）
   - 2026年1〜5月（東京）: 前年同期比▲31%（6人 vs 推計8.75人）

2. **事故件数は7万件前後で横ばい**が続いているが、2025年に約67,450件（推計）と若干減少

3. **2026年4月の青切符制度**
   - 第1ヶ月: 2,147件（想定の半分）
   - 指導警告: 135,855件（認知不足を示す）
   - 効果出現には**6ヶ月〜1年の定着期間**が必要との見方

4. **追い越し間隔規定（5月23日施行）は施行直後**であり、2026年下半期以降に効果を評価すべき

5. **ヘルメット着用率21.2%は依然として低い**が、死者数減少には寄与していると考えられる

### データの限界・留意事項

| 制約 | 内容 |
|------|------|
| 2026年全国月別データ | 未公表（警察庁は通常翌月中旬〜月末に公開） |
| 2025年全国自転車死者数 | 確定値は2026年3月公表済みだが本調査では推計値使用 |
| 東京2026YTD | 警視庁日報ページの解釈に基づく（公式集計表ではない） |
| 地域差 | 東京都データは全国の代表値ではない（東京は自転車利用率・警察密度が特殊） |
| 因果関係 | 死者数減少と法改正の因果関係は統計的に未検証 |
