## 3. ゴールド系：2倍・3倍ブル商品

### 3-1. UGL（ProShares Ultra Gold 2x）✅ SBI証券取扱確認

**ゴールド系で最も使いやすい2倍ブルETF**

| 項目 | 内容 |
|------|------|
| ティッカー | UGL |
| 上場市場 | NYSE Arca（米国） |
| 連動目標 | Bloomberg Gold Subindex の日次2倍 |
| レバレッジ | **2倍（ブル）** |
| 経費率（TER） | **0.95%/年** |
| SBI証券取扱 | **○（確認済み。NYSE上場 $49.76）** |
| 米国ETF買付手数料 | **実質無料**（キャッシュバック） |
| NISA対応 | **不可**（レバレッジETFはNISA対象外） |
| 受渡日数 | T+2 |
| 税制 | 株式等（申告分離20.315%） |
| 株式損益との通算 | **可** |

#### コスト算出（SOFR = 3.63%）

```
【ETF内部コスト構造】

スワップ埋込コスト = SOFR × (レバレッジ - 1)
                  = 3.63% × (2 - 1) = 3.63%/年  [推計]

経費率          = 0.95%/年
実質年率コスト  = 0.95% + 3.63% = 4.58%/年  [推計]

ボラティリティドラッグ（ボラドラッグ）
  D = 0.5 × L × (L-1) × σ²
  = 0.5 × 2 × 1 × 0.162²
  = 0.5 × 2 × 1 × 0.02624
  ≈ 2.62%/年

合計コスト目安  = 4.58% + 2.62% = 約7.2%/年  [推計]
```

#### 保有期間別コスト実額（元本100万円・横ばい想定）

| 保有期間 | 実質コスト（スワップ埋込+経費率のみ） | ボラドラッグ込み概算 |
|---------|--------------------------------------|---------------------|
| 1ヶ月 | 約3,800円 | 約6,000円 |
| 3ヶ月 | 約11,450円 | 約18,000円 |
| 1年 | 約45,800円 | 約72,000円 |

**活用ポイント：**
- SBI証券で特定口座対応。株式損益通算が可能
- ボラドラッグが年率2.62%かかるため中長期保有には不向き
- 金価格の短中期の上昇トレンドを捉えるのに適している
- **2倍止まりであり、3倍は達成できない**

### 3-3. SBI証券 金CFD（任意レバレッジ） ✅

**ゴールドに3倍以上をかけるにはSBI証券ではこれのみ**

| 項目 | 内容 |
|------|------|
| 商品種別 | 店頭CFD（差金決済）|
| 連動対象 | XAU/USD（金国際価格） |
| 最大レバレッジ | **20倍**（証拠金率5%） |
| 3倍に設定する場合 | 証拠金の3倍のポジションを建てる（証拠金率33%相当） |
| オーバーナイト金利 | **非公開**（業界標準から推計：年率4.4〜4.9%） |
| スプレッド | **非公開**（変動制） |
| 受渡日数 | **T+0（即時決済）** |
| ボラドラッグ | **0%**（日次リバランス型ではない） |
| 税制 | 先物取引に係る雑所得等（申告分離20.315%） |
| 株式損益との通算 | **不可**（CFD・先物間のみ通算可） |

#### 3倍設定時のコスト算出

```
オーバーナイト金利（年率・推計） ≈ SOFR + 金リース金利 + 業者マージン
                              = 3.63% + 0.3% + 0.5〜1.0%
                              ≈ 4.4〜4.9%/年  [推計]

CFDは「想定元本（名目ポジション）」に金利がかかる。
3倍設定時：元本100万円 × 3倍 = 想定元本300万円

保有コスト（年） = 300万円 × 4.7% = 141,000円/年
元本比コスト    = 141,000円 / 100万円 = 14.1%/年  [推計]
```

> ⚠️ **CFDのレバレッジとコストの関係は直線的：** 3倍設定では金利は3倍かかる（元本比）。ETFのスワップ埋込（(L-1)倍）と異なり、CFDは「借りたポジション全体」に金利がかかる。ただしボラドラッグがゼロのため、高ボラ環境下ではETFより実質コストが低い場合がある。

#### 保有期間別コスト実額（元本100万円・3倍設定・横ばい想定）

| 保有期間 | オーバーナイト金利のみ（推計）| ボラドラッグ |
|---------|------------------------------|------------|
| 1ヶ月 | 約11,750円 | 0円 |
| 3ヶ月 | 約35,250円 | 0円 |
| 1年 | 約141,000円 | 0円 |

**活用ポイント：**
- 3倍以上のレバレッジはSBI証券では**金CFDのみ**で実現可能
- T+0決済のため緊急時の即時クローズが可能
- コストは非透明なため**事前見積もりが困難**（実測必須）
- 金利（想定元本ベース）が大きいため、長期保有より短中期向き

## 4. ボンド系：2倍・3倍ブル商品

### 4-1. TMF（Direxion Daily 20+ Year Treasury Bull 3X）✅ SBI証券取扱確認

**ボンド系でSBI証券が取り扱う唯一の3倍ブルETF**

| 項目 | 内容 |
|------|------|
| ティッカー | TMF |
| 上場市場 | NYSE（米国） |
| 連動目標 | ICE U.S. Treasury 20+ Year Bond Index の日次3倍 |
| レバレッジ | **3倍（ブル）** |
| 経費率（TER） | **0.95%/年**（2026年9月まで上限キャップあり） |
| SBI証券取扱 | **○（確認済み）** |
| 米国ETF買付手数料 | **実質無料**（キャッシュバック） |
| NISA対応 | **不可** |
| 受渡日数 | T+2 |
| 税制 | 株式等（申告分離20.315%） |
| 株式損益との通算 | **可** |

#### コスト算出（SOFR = 3.63%）

```
スワップ埋込コスト = SOFR × (レバレッジ - 1)
                  = 3.63% × (3 - 1) = 7.26%/年  [推計]

経費率          = 0.95%/年
実質年率コスト  = 0.95% + 7.26% = 8.21%/年  [推計]

ボラドラッグ
  D = 0.5 × 3 × 2 × 0.125²
  = 0.5 × 3 × 2 × 0.015625
  = 4.69%/年

合計コスト目安  = 8.21% + 4.69% = 約12.9%/年  [推計]
```

> ⚠️ **TMFのスワップコストはETFの表面上の経費率（0.95%）には含まれていない。** Direxionのプロスペクタスは「スワップファイナンスコストは経費率キャップの算定から除外される」と明記している。本当のコストを理解するには経費率+スワップ埋込の合算が必須。

#### 保有期間別コスト実額（元本100万円・横ばい想定）

| 保有期間 | 実質コスト（スワップ+経費率） | ボラドラッグ込み概算 |
|---------|------------------------------|---------------------|
| 1ヶ月 | 約6,840円 | 約10,750円 |
| 3ヶ月 | 約20,525円 | 約32,250円 |
| 1年 | 約82,100円 | 約129,000円 |

**活用ポイント：**
- 長期金利が**低下する局面**でのリターンが期待できる
- 金利上昇局面ではパフォーマンス悪化（価格下落）＋保有コスト増大（スワップコスト上昇）の**二重苦**
- 利下げ局面（現在が該当する可能性）では「価格上昇益 + コスト低下」のダブルメリット
- 株式損益との通算が可能で、税務効率が良い
- SBI証券で特定口座対応（確定申告が容易）

## 6. 手数料の仕組み：3種類の商品形態別

### 6-1. 米国ETF（UGL・TMF・UBT）

米国ETFの手数料体系は以下の通り。

```
【購入時コスト】
① 買付手数料：SBI証券では実質無料（キャッシュバック）
② 為替スプレッド：通常25銭/ドル（住信SBIネット銀行積立で最低6銭）

【保有中コスト】
③ 経費率（TER）：自動的に基準価格から差し引かれる（UGL/TMF/UBT = 0.95%/年）
④ スワップ埋込コスト：表示されないが内部で発生（SOFR×（L-1））
⑤ ボラティリティドラッグ：日次リバランスによる構造的損失

【売却時コスト】
⑥ 売却手数料：約定代金の0.495%（上限22ドル）
⑦ 為替スプレッド（同上）
```

**見えないコスト（③〜⑤）が重要：**  
表面の「買付手数料無料」だけに目が向きやすいが、保有中の③スワップ埋込と⑤ボラドラッグが年率コストの大部分を占める。

### 6-2. 国内ETF×信用取引（1540）

```
【購入時コスト】
① 買付手数料：東証ETFの通常売買手数料（SBI証券では0円〜）
② 信用口座開設：無料

【保有中コスト】
③ 信用買い金利：年率2.80%（制度信用・日割計算）
④ 信託報酬：0.44%/年（自動差引）
⑤ ボラドラッグ：0%（現物ETFのため）

【売却時コスト】
⑥ 売却手数料：SBI証券の株式売買手数料（100万円以内なら無料）
⑦ 品貸料（一般信用の場合）：変動あり
```

### 6-3. 店頭CFD（SBI金CFD）

```
【建玉時コスト】
① 手数料：なし（スプレッドに含む）
② スプレッド：非公開・変動制

【保有中コスト（毎日課金）】
③ オーバーナイト金利 = 想定元本 × 日次金利率（SOFR+金リース+マージン）
  = 想定元本 × 約4.4〜4.9%/365 × 保有日数  [推計]
④ ボラドラッグ：0%（日次リバランスなし）

【決済時コスト】
⑤ スプレッド（反対売買時）：非公開
```

**CFDの最重要注意点：**
- オーバーナイト金利は「**想定元本（名目ポジション）**」にかかる（元本×レバレッジ）
- 3倍設定なら100万円の元本で300万円の名目ポジション→金利も300万円分
- SBI証券は金利率・スプレッドを**非公開**。実際のコストは口座明細での実測が必要

## 7. 商品選択フレームワーク

### 目的別選択ガイド

```
【ゴールドへの投資目的】
│
├── 2倍レバレッジ・コスト最小・長期保有
│   └─ ① 1540×信用買い（年率3.24%・ボラドラッグゼロ）
│      ※ ただし最大2倍止まり
│
├── 2倍レバレッジ・株式損益通算・SBI完結
│   └─ ② UGL（年率約7.2%・株式損益通算可）
│
├── 3倍以上・SBI完結・短中期
│   └─ ③ SBI金CFD（3倍設定・T+0即時・年率約14%[推計]）
│      ⚠️ コスト非透明・ストップロス必須
│
└── 3倍・欧州ETP（3GOL）→ 日本から購入不可

【ボンドへの投資目的】
│
├── 3倍ブル・利下げ局面・SBI完結
│   └─ ④ TMF（年率約12.9%・株式損益通算可・SBI特定口座対応）
│      ⚠️ 高コスト。利下げ局面の短中期保有に限定推奨
│
├── 2倍ブル・コスト抑制
│   └─ ⑤ UBT（年率約6.1%）→ SBI取扱要確認
│      SBI証券の米国ETF検索で「UBT」を検索して取扱確認を
│
└── 3倍超・T+0・高倍率
    └─ IG証券 米国債先物CFD → SBI証券では取扱なし
```

### 利下げ・利上げシナリオ別の影響

| シナリオ | ゴールドへの影響 | ボンドへの影響 |
|---------|----------------|--------------|
| **利下げ進行**（現在の基本シナリオ）| SOFR低下→UGLのスワップコスト軽減。CFD金利も低下 | TMFに最もプラス（価格上昇＋コスト低下のダブルメリット）|
| **横ばい（金利据置）** | 現状コスト継続 | 現状コスト継続（年率12.9%の重荷）|
| **利上げ再開** | コスト上昇（UGL・CFD両方）| TMFに最悪（価格下落＋コスト増大の二重苦）|
